Linux 的根目录（`/`）下的文件和目录各自有特定的用途，以下是常见目录的含义：

| 目录 | 作用 |
| --- | --- |
| `/bin` | 存放基本的用户二进制可执行文件（如 `ls`
, `cp`
, `mv`
, `cat`
 等），系统可在单用户模式下使用。 |
| `/boot` | 存放启动加载器（如 `grub`
）及 Linux 内核（`vmlinuz`
）、初始化 `ramdisk`
（`initrd`
）等。 |
| `/dev` | 存放设备文件（如 `sda`
 代表磁盘，`tty`
 代表终端，`null`
 代表空设备）。 |
| `/etc` | 存放系统配置文件（如 `passwd`
、`shadow`
、`fstab`
、`hosts`
 等）。 |
| `/home` | 存放普通用户的主目录，如 `/home/user1`
。 |
| `/lib` | 存放系统运行所需的基本共享库（如 `libc.so`
）和内核模块。 |
| `/media` | 挂载点目录，自动挂载可移动设备（如 U 盘、CD-ROM）。 |
| `/mnt` | 临时挂载目录，可用于手动挂载文件系统。 |
| `/opt` | 用于安装附加的第三方软件（如 `/opt/google/chrome`
）。 |
| `/proc` | 伪文件系统，存放内核及进程相关信息（如 `/proc/cpuinfo`
、`/proc/meminfo`
）。 |
| `/root` | `root`
 用户的主目录，类似普通用户的 `/home/username`
。 |
| `/run` | 存放系统运行时产生的临时文件，如进程 ID 文件（`*.pid`
）。 |
| `/sbin` | 存放系统管理命令（如 `fdisk`
, `shutdown`
, `ifconfig`
），普通用户一般无权限执行。 |
| `/srv` | 存放系统提供的服务数据，如 Web 服务器的网页数据（`/srv/www`
）。 |
| `/sys` | 伪文件系统，存放设备和驱动相关信息（如 `/sys/block`
）。 |
| `/tmp` | 存放临时文件，系统重启后会清空该目录。 |
| `/usr` | 存放系统软件、库、文档等，通常是只读的。 |
| `/usr/bin` | 存放非必要的用户二进制文件，如 `vim`
, `gcc`
。 |
| `/usr/lib` | 存放非必要的库文件。 |
| `/usr/local` | 用于本地安装的软件，不受包管理器控制（如 `usr/local/bin`
 存放本地安装的可执行文件）。 |
| `/var` | 存放变化的数据，如日志文件（`/var/log`
）、邮件队列（`/var/mail`
）、缓存文件等。 |
