# 流水线部署快速方法

通过手动修改 Jenkins builds 目录中的文件，模拟流水线构建成功状态。

## 操作步骤

进入到 jenkins 容器的 `/var/jenkins_home/jobs/cloud-manage-public/builds` 目录

添加以下的内容

```bash
#/var/jenkins_home/jobs/cloud-manage-public/builds
jenkins: "slave"
jenkins/label: "jnlp"
Finished: SUCCESS
mvn -B -DskipTests clean package

#可以跑个test  shell脚本如果成功，里面jobs  test目录会有以下三个参数
jenkins: "slave"
jenkins/label: "jnlp"
Finished: SUCCESS
#手动添加
mvn -B -DskipTests clean package
```
