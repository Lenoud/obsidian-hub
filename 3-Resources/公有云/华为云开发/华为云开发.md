# 华为云 Python SDK 安装指南

华为云 Python SDK 安装与配置，用于通过 Python 管理华为云资源。

SDK 中心：https://console.huaweicloud.com/apiexplorer/#/sdkcenter?language=python

## SDK 库安装
[https://console.huaweicloud.com/apiexplorer/#/sdkcenter?language=python](https://console.huaweicloud.com/apiexplorer/#/sdkcenter?language=python)

## sdk库安装
```bash
#https://support.huaweicloud.com/sdkreference-mpc/mpc_05_0076.html
#安装华为云最新版本 SDK 的依赖库，包括弹性云服务、虚拟私有云、镜像服务、容器云引擎、云数据库的 python

pip3 install -U huaweicloud-sdk-python -i https://mirrors.aliyun.com/pypi/simple/
pip3 install -U huaweicloudsdkcore  -i https://mirrors.aliyun.com/pypi/simple/
pip3 install -U huaweicloudsdkmpc  -i https://mirrors.aliyun.com/pypi/simple/
pip3 install -U huaweicloudsdkcce  -i https://mirrors.aliyun.com/pypi/simple/
pip3 install -U huaweicloudsdkecs  -i https://mirrors.aliyun.com/pypi/simple/
pip3 install -U huaweicloudsdkims -i https://mirrors.aliyun.com/pypi/simple/
pip3 install -U huaweicloudsdkdms -i https://mirrors.aliyun.com/pypi/simple/
pip3 install -U huaweicloudsdkevs -i https://mirrors.aliyun.com/pypi/simple/
pip3 install -U huaweicloudsdkas -i https://mirrors.aliyun.com/pypi/simple/
pip3 install -U huaweicloudsdkvpc -i https://mirrors.aliyun.com/pypi/simple/
pip3 install -U huaweicloudsdkims -i https://mirrors.aliyun.com/pypi/simple/
```
