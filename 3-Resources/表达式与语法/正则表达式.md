# 正则表达式

正则表达式相关技术笔记。

**1. 字符匹配：**

+ **普通字符：** 正则表达式中的普通字符（例如字母、数字和标点符号）会直接匹配相应的字符。例如，正则表达式**hello**将匹配字符串中的 "hello"。
+ **特殊字符：** 正则表达式中有一些特殊字符需要转义，如 **\**, **.**, *, **?**, **+**, **|**, **()**, **[]**, **{}**, 等。这些字符具有特殊含义，需要用反斜杠 **\** 转义，以匹配字面值。

**2. 字符类和范围：**

+ **[ ]**：字符类，用于匹配其中的任何一个字符。例如，**[aeiou]** 匹配任何一个元音字母。
+ **[a-z]**：字符范围，用于匹配指定范围内的字符。例如，**[0-9]** 匹配任何一个数字。

**3. 量词：**

+ *：匹配前一个元素零次或多次。
+ **+**：匹配前一个元素一次或多次。
+ **?**：匹配前一个元素零次或一次。
+ **{n}**：精确匹配前一个元素 n 次。
+ **{n,}**：匹配前一个元素至少 n 次。
+ **{n,m}**：匹配前一个元素至少 n 次，最多 m 次。

**4. 锚点：**

+ **^**：匹配字符串的开头。
+ **$**：匹配字符串的结尾。
+ **\b**：匹配单词边界。

**5. 特殊字符类：**

+ **\d**：匹配任何数字字符（等同于 **[0-9]**）。
+ **\w**：匹配任何单词字符（字母、数字、下划线）。
+ **\s**：匹配任何空白字符（空格、制表符、换行符等）。
