# Markdown语法表

Markdown语法表相关技术笔记。

## Markdown语法表
| **语法** | **说明** |
| :--- | :--- |
| # | 标题 |
| ** | 粗体 |
| * | 斜体 |
| *** | 粗体加斜体 |
| ~~ | 删除线 |
| `<font>` | 字体设置 |
| `<font size="">` | 字体大小设置 |
| `<font color="">` | 字体颜色设置 |
| ----- | 分割线 |
| `<u>` | 下划线 |
| > 引用文字 | 引用 |
| [图片alt](图片地址 ''图片title'') | 图片 |
| [超链接名]( 超链接地址 "超链接title") | 超链接 |
| * 或 + 或 - | 无序列表 |
| 数字 + . | 有序列表 |
| ` | 单行代码 |
| ``` | 代码块 |
| flow & | 流程图 |
| <sup></sup> | 上标 |
| <sub></sub> | 下标 |
