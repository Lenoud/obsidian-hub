# Linux 登录日志清理

清除 Linux 系统中的历史命令记录和登录日志文件。

## 操作命令

```bash
history -c
echo "" > /var/log/wtmp
echo "" > /var/log/btmp
echo "" > ~/.bash_history
```

## 说明

| 文件 | 说明 |
|------|------|
| `~/.bash_history` | 当前用户的历史命令记录 |
| `/var/log/wtmp` | 成功登录记录（`last` 命令读取） |
| `/var/log/btmp` | 失败登录记录（`lastb` 命令读取） |
