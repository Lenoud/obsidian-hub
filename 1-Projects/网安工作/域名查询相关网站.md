# 域名查询相关网站

域名 WHOIS 查询、历史解析记录、IP 反查等常用工具网站。

## WHOIS 查询

- 美橙互联 WHOIS：https://whois.cndns.com/
- 站长之家 WHOIS：https://whois.chinaz.com/
- WHOIS.com：https://www.whois.com/
- ICANN Lookup：https://lookup.icann.org/

## 历史解析与 IP 反查

- SecurityTrails：https://securitytrails.com/ — 域名解析历史与 IP 绑定信息
- CompleteDNS：https://completedns.com/ — DNS 历史记录查询
- OSINT.sh：https://osint.sh/ — 综合开源情报工具
- WhoisXML API DNS History：https://dns-history.whoisxmlapi.com/ — IP 反查域名
- DNSGrep：https://www.dnsgrep.cn/ — 子域查找与 IP 反查
