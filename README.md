# Obsidian Knowledge Hub

基于 [PARA 方法论](https://fortelabs.com/blog/para/) 组织的个人技术知识库，使用 [Obsidian](https://obsidian.md) 管理。

## 目录结构

```
1-Projects/     活跃项目（网安工作、skyris）
2-Areas/        持续技术领域
  容器与编排/     Kubernetes / Docker / Containerd / Istio / CICD
  Linux运维/     操作系统 / 存储 / 防火墙 / 虚拟化
  数据库/        MySQL / PostgreSQL / Redis / MongoDB / SQLServer / MariaDB
  网络技术/      Cisco / 软路由 / 内网穿透 / 路由交换
  自动化运维/    Ansible / Shell 脚本
  开发编程/      Git / API / Python / Golang
  服务部署/      Nginx / Prometheus / RabbitMQ / Clash
  Windows/       使用技巧 / bat 脚本 / Office
3-Resources/    参考资料（面试题 / 正则 / AI / 公有云 / 赛事参考）
4-Archive/      归档（学校课程 / 毕业设计 / OpenStack / 旧笔记）
```

## 内容统计

约 1000+ 篇 Markdown 技术笔记，涵盖运维、开发、网络、安全等领域。

## 使用方式

1. Clone 本仓库
2. 用 Obsidian 打开仓库目录作为 Vault
3. 安装推荐插件：`obsidian-git`

## 图片说明

所有图片使用 Obsidian Wiki 链接格式 `![[filename.png]]`，附件存放在 `attachments/` 目录。
