# Cloudkitty

Cloudkitty相关技术笔记。

Cloudkitty服务安装与使用
安装完毕后，启用hashmap评级模块，接着创建volume_thresholds组，创建服务匹配规则volume.size，并设置每GB的价格为0.01，接下来对应大量数据设置应用折扣，在组volume_thresholds中创建阈值，设置若超过50GB的阈值，应用2%的折扣(0.98)

```shell
[root@controller  ~]# iaas-install-cloudkitty.sh
[root@controller  ~]# openstack rating module enable hashmap
+---------+---------+----------+
| Module  | Enabled | Priority |
+---------+---------+----------+
| hashmap | True    |        1 |
+---------+---------+----------+
[root@controller  ~]# openstack rating hashmap service create volume.size
+-------------+--------------------------------------+
| Name        | Service ID                           |
+-------------+--------------------------------------+
| volume.size | 66dfac8c-80ef-4a87-8d3b-bca45c90725c |
+-------------+--------------------------------------+
[root@controller  ~]# openstack rating hashmap group create volume_thresholds
+-------------------+--------------------------------------+
| Name              | Group ID                             |
+-------------------+--------------------------------------+
| volume_thresholds | b5d95ff0-68f4-44fc-85ed-dc4f066b4d42 |
+-------------------+--------------------------------------+
[root@controller  ~]# openstack rating hashmap mapping create -s 66dfac8c-80ef-4a87-8d3b-bca45c90725c -g b5d95ff0-68f4-44fc-85ed-dc4f066b4d42 -t flat 0.01
+--------------------------------------+-------+------------+------+----------+--------------------------------------+--------------------------------------+------------+
| Mapping ID                           | Value | Cost       | Type | Field ID | Service ID                           | Group ID                             | Project ID |
+--------------------------------------+-------+------------+------+----------+--------------------------------------+--------------------------------------+------------+
| 4ab00401-de93-4449-a2f5-fb31f90d3626 | None  | 0.01000000 | flat | None     | 66dfac8c-80ef-4a87-8d3b-bca45c90725c | b5d95ff0-68f4-44fc-85ed-dc4f066b4d42 | None       |
+--------------------------------------+-------+------------+------+----------+--------------------------------------+--------------------------------------+------------+

[root@controller  ~]# openstack rating hashmap threshold create -s 66dfac8c-80ef-4a87-8d3b-bca45c90725c -g b5d95ff0-68f4-44fc-85ed-dc4f066b4d42 -t rate 50 0.98
+--------------------------------------+-------------+------------+------+----------+--------------------------------------+--------------------------------------+------------+
| Threshold ID                         | Level       | Cost       | Type | Field ID | Service ID                           | Group ID                             | Project ID |
+--------------------------------------+-------------+------------+------+----------+--------------------------------------+--------------------------------------+------------+
| ea333c46-4f47-4431-b5f0-022b7a93eb05 | 50.00000000 | 0.98000000 | rate | None     | 66dfac8c-80ef-4a87-8d3b-bca45c90725c | b5d95ff0-68f4-44fc-85ed-dc4f066b4d42 | None       |
+--------------------------------------+-------------+------------+------+----------+--------------------------------------+--------------------------------------+------------+
```
