# 结构思考力
结构思考力是一种通过 结构化框架 整理信息、分析问题和表达观点的思维方式，目标是让思考更清晰、表达更有力。

**核心特点**

拆解复杂问题：将模糊问题分解为可操作的模块。

逻辑化表达：避免碎片化信息，用结构提升说服力。

高效决策：通过框架快速抓住关键因素。

# 复盘
### 本次学习的复盘
![[1747029962321-25d8b429-7806-4ef9-97db-daefaab8268e.png]]

为什么要复盘：从经验中学习

#### 复盘的四个价值
1. 知其然，知其所以然

知道成功与失败的理由

2. 传承经验，提升能力
3. 不再犯同样的错误

同样一个问题，不要犯第二次，A公司犯过的错，同步给B公司，B公司应该避免重复犯下A公司以往的过错，避免资源浪费

4. 总结规律，固化流程

每次复盘都是对于当前制度重新审视的一个过程

![[1746691309358-f71e1598-31b2-4baa-b1ab-9f247f875c8e.png]]

#### 美军案例
目标：将装备运送到海滩地区

结果：比原来晚了数小时

原因：为了抢速度，车轮未困扎防滑链，陷入沙滩中

AAR后得出结论：

+ 未来的任务中一定会按照标准化流程作业
+ 对于整个美军而言，这次问题给了沙滩/沙漠作战提供了范例

复盘的目的：

1. 学习为导向

目的：让个人和团队通过经历得到学习，而不是以批判、追究为目的

2. 结构化流程和逻辑
3. 复盘更适合以团队形式进行

应该调动整个团队进行复盘，从执行层，自下而上反馈信息才是最真实有效的

### 如何做好复盘
复盘，心态很重要

组织复盘的人，需要明确本次复盘的原因：只是希望从这个过程中看到不足之地，以学习、进步为目的进行复盘

# 团队协作的五大障碍
## 五大障碍
1. 缺乏信任（团队成员不愿展现脆弱性，害怕被评判）

2. 惧怕冲突（因缺乏信任，团队回避必要的辩论）

3. 欠缺投入（因惧怕冲突，决策时模棱两可）

4. 逃避责任（因投入不足，成员不愿互相监督）

5. 无视结果（因责任缺失，个人利益凌驾于团队目标）

![[1746686217209-2fda0f20-5657-49f6-96dd-a8f5d16aef92.png]]

## 四个原则
1. 领导者要承担起协作的责任

2. 树立清晰的价值观

3. 令人对开会决定紧张又期待

4. 协作需要刻意的练习
