# MySQL 触发器

MySQL 触发器是在对表执行 INSERT、UPDATE、DELETE 操作时自动执行的 SQL 语句。

## 创建触发器

```sql
-- 创建示例：在 product 表插入数据后，自动记录时间到 time 表
CREATE TRIGGER trig1 AFTER INSERT ON product
FOR EACH ROW INSERT INTO time(t) VALUES(now());

-- 验证
INSERT INTO product VALUES(1000, "test", 1000, "cc10");
SELECT * FROM time;
```

## 查看触发器

```sql
SHOW TRIGGERS\G;
SHOW CREATE TRIGGER trig1\G;
SELECT * FROM information_schema.triggers\G;
SELECT * FROM information_schema.triggers WHERE trigger_name = "trig1"\G;
```

## 删除触发器

```sql
DROP TRIGGER trig1;
```

## 说明

| 语法元素 | 说明 |
| --- | --- |
| `BEFORE` / `AFTER` | 触发时机：操作前或操作后 |
| `INSERT` / `UPDATE` / `DELETE` | 触发事件 |
| `FOR EACH ROW` | 行级触发器，每行数据变更都触发 |
