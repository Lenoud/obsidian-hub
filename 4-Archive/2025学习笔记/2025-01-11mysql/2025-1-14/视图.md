# MySQL 视图

MySQL 视图（View）是虚拟表，基于 SQL 查询结果定义，简化复杂查询。

## 创建视图

```sql
CREATE VIEW view_student_course AS
(SELECT * FROM student LEFT JOIN course ON student.id = course.sid);

SELECT * FROM view_student_course;
```

## 查看视图定义

```sql
SHOW CREATE VIEW view_student_course\G;
```

## 修改视图

```sql
ALTER VIEW view_student_course AS (SELECT * FROM test2.product);
```

## 删除视图

```sql
DROP VIEW view_student_course;
```

## 说明

| 操作 | 语法 | 说明 |
| --- | --- | --- |
| 创建 | `CREATE VIEW 视图名 AS 查询语句` | 基于查询结果创建虚拟表 |
| 查看 | `SHOW CREATE VIEW 视图名` | 查看视图定义 |
| 修改 | `ALTER VIEW 视图名 AS 查询语句` | 修改视图的查询定义 |
| 删除 | `DROP VIEW 视图名` | 删除视图 |
