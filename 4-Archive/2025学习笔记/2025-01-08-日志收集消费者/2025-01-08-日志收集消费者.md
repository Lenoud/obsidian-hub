# Kafka 日志收集消费者

使用 Python 多进程消费 Kafka 多分区日志数据，清洗后写入 MySQL 数据库。

## 完整代码

```python
"""
@author: wy
@file: consumer.py
@time: 2025/1/7 13:47
"""
from kafka import KafkaConsumer
from multiprocessing import Process, current_process
import pymysql
import time
import json

conn = pymysql.connect(
    host="192.168.100.120",
    user="root",
    passwd="000000",
    port=3306,
    db="log_consumer"
)
cur = conn.cursor()

month_abbr_to_num = {
    'Jan': '01', 'Feb': '02', 'Mar': '03', 'Apr': '04',
    'May': '05', 'Jun': '06', 'Jul': '07', 'Aug': '08',
    'Sep': '09', 'Oct': '10', 'Nov': '11', 'Dec': '12'
}

def consume_kafka_partition(topic, group_id, partition):
    """进程执行的函数，用于消费指定分区的Kafka消息"""
    consumer = KafkaConsumer(
        group_id=group_id,
        bootstrap_servers=['192.168.100.150:9092','192.168.100.151:9092','192.168.100.152:9092'],
        auto_offset_reset='earliest',
        enable_auto_commit=True,
        auto_commit_interval_ms=5000,
        value_deserializer=lambda x: x.decode('utf-8')
    )
    consumer.subscribe(topic)

    for message in consumer:
        print(f"进程 {current_process().name} 消费到来自分区 {partition} 的消息: {message.value}")

        json_format = json.loads(message.value)
        message_data = json_format["message"]
        data_list = message_data.split()
        date = data_list[3][1:].split("/")
        time_parts = date[2].split(":")
        year = time_parts[0]
        month = month_abbr_to_num[date[1]]
        day = date[0]
        shi = time_parts[1]
        fen = time_parts[2]
        miao = time_parts[3]
        log_datetime = f"{year}-{month}-{day} {shi}:{fen}:{miao}"

        ipaddr = data_list[0]
        liuliang = data_list[9]
        company = "huawei"
        shengfen = "中国"

        result = [ipaddr, shengfen, company, log_datetime, liuliang]
        sql = "INSERT INTO log values(%s,%s,%s,%s,%s)"
        cur.execute(sql, result)
        conn.commit()

if __name__ == "__main__":
    topic = "nginxlog"
    group_id = "message_group3"
    partitions = [0, 1, 2]

    processes = []
    for partition in partitions:
        p = Process(target=consume_kafka_partition, args=(topic, group_id, partition))
        p.name = f"Consumer-Partition-{partition}"
        p.start()
        processes.append(p)

    for p in processes:
        p.join()
```

## 说明

| 组件 | 说明 |
| --- | --- |
| `KafkaConsumer` | Kafka 消费者，连接三节点集群 |
| `multiprocessing.Process` | 每个分区启动一个进程并行消费 |
| `pymysql` | 将清洗后的日志数据写入 MySQL |
| `auto_offset_reset='earliest'` | 从最早的消息开始消费 |
| `auto_commit_interval_ms=5000` | 每 5 秒自动提交偏移量 |
