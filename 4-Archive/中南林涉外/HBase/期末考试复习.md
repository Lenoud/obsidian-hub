

### 命令书写题
向命名空间A的表user添加一个单元格，单元格的行键、列族、列标识、和数据分别是001、person、name和zhangsan

```powershell
put ‘A:user’,’001’,’person:name’,’zhangsan’
```

查看命名空间default中表teachername的信息

```powershell
desc ‘defalut:teachername’
```

查看命名空间hbase的表

```powershell
list_namespace_tables  ‘hbase’
```

使用deleteall命令，删除命名空间default中表teacher_info的数据，指定行键为user005

```powershell
deleteall ‘default:teacher_info’,’user005’
```

判断命名空间school是否存在表student_info

```powershell
exists ‘school:student_info’
```

使用scan命令查询命名空间default中的表user_info列族name的数据

```powershell
scan  ‘default:user_info’,{COLUMNS=>’name’}
```
