### 整理后的 `os` 常用命令及实例代码
#### **1. 文件/目录操作**
1. **获取当前工作目录**: `os.getcwd()`

```python
python

复制代码
import os
print("当前工作目录:", os.getcwd())
```

2. **改变当前工作目录**: `os.chdir(path)`

```text
python

复制代码
os.chdir("/path/to/directory")
print("切换后工作目录:", os.getcwd())
```

3. **列出目录内容**: `os.listdir(path)`

```text
python

复制代码
print("目录内容:", os.listdir("."))
```

4. **创建单层目录**: `os.mkdir(path)`

```text
python

复制代码
directory = "example_dir"
if not os.path.exists(directory):
    os.mkdir(directory)
    print(f"目录 '{directory}' 创建成功！")
```

5. **递归创建多层目录**: `os.makedirs(path)`

```text
python

复制代码
nested_dir = "example_dir/sub_dir"
if not os.path.exists(nested_dir):
    os.makedirs(nested_dir)
    print(f"多层目录 '{nested_dir}' 创建成功！")
```

6. **删除文件**: `os.remove(path)`

```text
python

复制代码
file_path = "example.txt"
if os.path.exists(file_path):
    os.remove(file_path)
    print(f"文件 '{file_path}' 删除成功！")
```

7. **删除空目录**: `os.rmdir(path)`

```text
python

复制代码
empty_dir = "empty_folder"
if os.path.exists(empty_dir):
    os.rmdir(empty_dir)
    print(f"空目录 '{empty_dir}' 删除成功！")
```

8. **递归删除目录树**: `os.removedirs(path)`

```text
python

复制代码
nested_dir = "example_dir/sub_dir"
if os.path.exists(nested_dir):
    os.removedirs(nested_dir)
    print(f"目录树 '{nested_dir}' 删除成功！")
```

9. **重命名文件或目录**: `os.rename(src, dst)`

```text
python

复制代码
old_name = "old_name.txt"
new_name = "new_name.txt"
if os.path.exists(old_name):
    os.rename(old_name, new_name)
    print(f"文件重命名为 '{new_name}'")
```

#### **2. 路径操作**
1. **拼接路径**: `os.path.join()`

```text
python

复制代码
path = os.path.join("folder", "file.txt")
print("拼接路径:", path)
```

2. **获取绝对路径**: `os.path.abspath(path)`

```text
python

复制代码
print("绝对路径:", os.path.abspath("file.txt"))
```

3. **判断路径是否存在**: `os.path.exists(path)`

```text
python

复制代码
print("路径是否存在:", os.path.exists("example.txt"))
```

4. **判断是否为目录**: `os.path.isdir(path)`

```text
python

复制代码
print("是否为目录:", os.path.isdir("example_dir"))
```

5. **判断是否为文件**: `os.path.isfile(path)`

```text
python

复制代码
print("是否为文件:", os.path.isfile("example.txt"))
```

6. **分离文件名和扩展名**: `os.path.splitext(path)`

```text
python

复制代码
file_name, file_ext = os.path.splitext("example.txt")
print("文件名:", file_name, "扩展名:", file_ext)
```

7. **获取目录名**: `os.path.dirname(path)`

```text
python

复制代码
print("目录名:", os.path.dirname("/path/to/example.txt"))
```

8. **获取文件名**: `os.path.basename(path)`

```text
python

复制代码
print("文件名:", os.path.basename("/path/to/example.txt"))
```

### 综合实例代码：文件夹创建与内容查看
```python
import os

# 指定目录路径
directory = "test_folder"

# 判断目录是否存在，不存在则创建
if not os.path.exists(directory):
    os.mkdir(directory)
    print(f"目录 '{directory}' 创建成功！")
else:
    print(f"目录 '{directory}' 已存在。")

# 列出目录内容
print("目录内容:", os.listdir(directory))

# 拼接新文件路径
file_path = os.path.join(directory, "example.txt")

# 创建文件并写入内容
with open(file_path, "w") as file:
    file.write("Hello, OS Library!")
    print(f"文件 '{file_path}' 创建并写入内容成功！")

# 获取文件大小
print("文件大小:", os.path.getsize(file_path), "字节")
```
