# 推送镜像脚本

推送镜像脚本相关技术笔记。

使用 `nerdctl` 将本地镜像标记并推送到 Harbor 私有仓库的 Bash 脚本。

## 说明

| 变量 | 值 | 说明 |
| :--- | :--- | :--- |
| HARBOR | 192.168.100.105 | Harbor 私有仓库地址 |
| TAG | jenkins-slave1252 | Harbor 项目名称 |
| VERSION | latest | 镜像标签版本 |

## 脚本内容

```bash
#!/bin/bash
HARBOR=192.168.100.105
TAG=jenkins-slave1252
VERSION=latest
#nerdctl -n k8s.io image tag jenkins/jenkins:jnlp   $HARBOR/$TAG/jenkins/jenkins:jnlp
nerdctl -n k8s.io image tag jenkins/jenkins $HARBOR/$TAG/gitlab/gitlab-ce:VERSION

#nerdctl -n k8s.io push $HARBOR/$TAG/jenkins/jenkins
nerdctl -n k8s.io push $HARBOR/$TAG/gitlab/gitlab-ce:VERSION
```
