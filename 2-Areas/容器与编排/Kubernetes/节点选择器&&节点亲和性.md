# 节点选择器与节点亲和性

节点选择器与节点亲和性相关技术笔记。

通过 nodeName、nodeSelector 以及亲和性/反亲和性规则控制 Pod 调度到指定节点。

## 直接指定节点

```yaml
      nodeName: k8s-worker-node1
```

## 指定节点标签

```yaml
      nodeSelector:
        app: web
```

## 节点标签管理

```bash
# 为节点打 label
kubectl label node k8s-master-node1 disktype=ssd

# 查询是否成功
kubectl get nodes --show-labels |grep  disktype=ssd

# 取消标签
kubectl label node k8s-master-node1 disktype-

# 查询哪些节点有这个键值对
kubectl get nodes --selector disktype=ssd
```

## nodeSelector 示例

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: my-label
  labels:
    env: test
spec:
  containers:
  - name: my-ssd
    image: nginx
  nodeSelector:   #部署到有此标签的节点上
    disktype: ssd  #标签名ssd
```
