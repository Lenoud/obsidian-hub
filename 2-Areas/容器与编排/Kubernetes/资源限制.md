# 资源限制

资源限制相关技术笔记。

通过 `spec.containers.resources` 为容器设置 CPU 和内存的请求（requests）与上限（limits），实现资源的合理分配。

```yaml
    spec:
      containers:
          resources:
            limits: #最大限制
              cpu: 500m
              memory: 1Gi
            requests: #最小限制
              cpu: 500m
              memory: 200Mi
```
