# 2048 游戏

经典的 2048 数字拼图游戏，通过 Docker 容器部署，资源占用极低。

## 部署

```yaml
version: "3.8"
services:
  game:
    restart: always
    container_name: demo-2048
    image: alexwhen/docker-2048
    ports:
    - 80:80
    volumes:
    - /etc/localtime:/etc/localtime:ro
#    cpus: 0.2
#    mem_limit: 30m
    deploy:
      resources:
        limits:
          cpus: '0.2'
          memory: 20M
    networks:
      demo_bridge:
        ipv4_address: 172.21.0.2

networks:
  demo_bridge:
    driver: bridge
    ipam:
      config:
        - subnet: 172.21.0.0/16
```

## 说明

- 访问地址：`http://<IP>:80`
- 端口映射：`80` -> `80`
- 资源限制：0.2 核 CPU、20M 内存
- 固定 IP：`172.21.0.2`（子网 `172.21.0.0/16`）
