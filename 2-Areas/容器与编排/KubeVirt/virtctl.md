# virtctl

virtctl相关技术笔记。

这是 virtctl 的一些常用命令说明：

1. addvolume：将一个卷添加到正在运行的虚拟机中。
2. completion：生成指定 shell 的自动补全脚本。
3. console：连接到虚拟机实例的控制台。
4. expose：将虚拟机实例、虚拟机或虚拟机实例副本集公开为新服务。
5. fslist：返回客户机上可用文件系统的完整列表。
6. guestfs：启动到 libguestfs pod 的 shell。
7. guestosinfo：返回有关操作系统的客户代理信息。
8. help：获取任何命令的帮助信息。
9. image-upload：将 VM 镜像上传到 DataVolume / PersistentVolumeClaim。
10. memory-dump：将正在运行的虚拟机的内存转储到 PVC。
11. migrate：迁移虚拟机。
12. migrate-cancel：取消虚拟机的迁移。
13. pause：暂停虚拟机。
14. permitted-devices：列出 VMI 的许可设备。
15. port-forward：将本地端口转发到虚拟机或虚拟机实例。
16. removevolume：从正在运行的虚拟机中删除一个卷。
17. restart：重启虚拟机。
18. scp：在虚拟机实例和本地计算机之间传输文件。
19. soft-reboot：软重启虚拟机实例。
20. ssh：打开到虚拟机实例的 SSH 连接。
21. start：启动虚拟机。
22. stop：停止虚拟机。
23. unpause：取消暂停虚拟机。
24. usbredir：将 USB 设备重定向到虚拟机实例。
25. userlist：返回客户机上已登录用户的完整列表。
26. version：打印客户端和服务器版本信息。
27. vmexport：导出 VM 卷。
28. vnc：打开到虚拟机实例的 VNC 连接。
