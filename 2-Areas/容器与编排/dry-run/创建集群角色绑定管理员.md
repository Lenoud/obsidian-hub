# 创建集群角色绑定管理员

创建集群角色绑定管理员相关技术笔记。

使用 `kubectl create` 命令创建 ServiceAccount 并绑定 cluster-admin 集群管理员权限。

## 操作命令

```bash
kubectl create serviceaccount jenkins-admin -n jenkins
kubectl create clusterrolebinding jenkins-admin --clusterrole=cluster-admin \
--serviceaccount=jenkins:jenkins-admin
```

## 说明

| 参数 | 说明 |
| :--- | :--- |
| `--serviceaccount=jenkins:jenkins-admin` | jenkins 命名空间下的 jenkins-admin ServiceAccount |
| `--clusterrole=cluster-admin` | 集群管理员角色 |

这两个命令会分别创建 ServiceAccount 和 ClusterRoleBinding，使该 SA 拥有集群管理员权限。
