# 镜像导入

镜像导入相关技术笔记。

使用 `ctr` 和 `nerdctl` 命令将本地 tar 格式的容器镜像导入到 Containerd 的 `k8s.io` 命名空间中。

## 说明

| 命令 | 用途 | 说明 |
| :--- | :--- | :--- |
| `ctr -n=k8s.io image import` | 导入镜像到 k8s.io 命名空间 | 适用于 `.tar` 格式镜像 |
| `nerdctl -n k8s.io image load -i` | 加载镜像到 k8s.io 命名空间 | 适用于 `.tar` 格式镜像 |

## 操作命令

```bash
#导入镜像
ctr -n=k8s.io image import dashboard.tar
#--namespace=k8s.io指定命名空间，不是必须，根据环境而定
nerdctl -n k8s.io image load -i images.tar
```
