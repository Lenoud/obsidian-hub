# MySQL 数据库配置修改

MySQL 常用配置参数调优，涉及表名大小写、InnoDB 缓冲池、日志等关键设置。

## 配置参数

在 `my.cnf` 的 `[mysqld]` 段添加以下配置：

```ini
# 设置数据库支持大小写（1=不区分，0=区分）
lower_case_table_names = 1
# 设置 InnoDB 缓冲池大小
innodb_buffer_pool_size = 4G
# 设置 log buffer 大小
innodb_log_buffer_size = 64M
# 设置 redo log 文件大小
innodb_log_file_size = 256M
# 设置 redo log 文件组数量
innodb_log_files_in_group = 2
```

## 说明

| 参数 | 值 | 说明 |
| --- | --- | --- |
| `lower_case_table_names` | `1` | 表名不区分大小写 |
| `innodb_buffer_pool_size` | `4G` | InnoDB 缓存索引和数据的缓冲池大小 |
| `innodb_log_buffer_size` | `64M` | 日志缓冲区大小 |
| `innodb_log_file_size` | `256M` | 每个 redo log 文件大小 |
| `innodb_log_files_in_group` | `2` | redo log 文件组中的文件数 |
