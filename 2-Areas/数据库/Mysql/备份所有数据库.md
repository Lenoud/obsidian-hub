# MySQL 备份所有数据库

使用 `mysqldump` 的 `--all-databases` 参数一次性导出所有数据库。

## 操作命令

```bash
mysqldump -uroot -p"000000" --all-databases > openstack.sql
ls -lh /root/
```

重装系统后使用 `source` 命令导入恢复：

```sql
source /root/openstack.sql
```

## 说明

| 参数 | 说明 |
| --- | --- |
| `--all-databases` | 导出 root 用户下所有数据库（注意 `all` 前是两个减号） |
| `-p` | 后跟 MySQL root 密码 |
