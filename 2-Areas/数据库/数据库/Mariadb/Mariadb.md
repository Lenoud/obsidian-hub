# MariaDB 版本对照

MariaDB 与 MySQL 版本的功能对应关系。

| MariaDB 版本 | 对应 MySQL 版本 | 说明 |
| --- | --- | --- |
| 10.7、10.6、10.5、10.4 | MySQL 8.0 | 支持窗口函数、CTE、增强 JSON 等 |
| 10.3 | MySQL 5.7 | 引入窗口函数，兼容性良好 |
| 10.2 | MySQL 5.6 | 保留 MySQL 5.6 特性，增加 MariaDB 优化 |
| 10.1、10.0 | MySQL 5.5 | 提供 MySQL 5.5 大多数功能 |
| 5.5 | MySQL 5.5 | 直接对应，建议使用更新版本 |
