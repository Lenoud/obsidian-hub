# MySQL 修改 root 用户密码

MySQL 中修改 root 用户密码的常用方法。

## 方法一：UPDATE 语句（MySQL 5.x）

```sql
UPDATE user SET Password = password('000000') WHERE User = 'root';
FLUSH PRIVILEGES;
```

## 方法二：SET PASSWORD

```sql
SET PASSWORD FOR 'root'@'%' = PASSWORD('123456');
```

## 方法三：GRANT 授权（MySQL 5.x）

```bash
mysql -uroot -p000000 -e "grant all privileges on *.* to root@'%' identified by '000000';"
```

## 方法四：GRANT ALL（MySQL 8.x）

```sql
GRANT ALL ON *.* TO 'root'@'%';
FLUSH PRIVILEGES;
```
