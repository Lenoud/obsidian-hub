# SQL 生成显示所有表名查询语句

自动生成包含字段名和注释的 SELECT 查询语句模板。

## SQL 语句

```sql
SELECT
    CONCAT('select ', t_info, ' from 替换表名;') AS 打印所有表名查询sql语句
FROM
    (SELECT GROUP_CONCAT(COLUMN_NAME, ' ', COLUMN_COMMENT) AS t_info
     FROM information_schema.COLUMNS
     WHERE table_name = '替换表名') t;
```

## 说明

| 函数 | 说明 |
| --- | --- |
| `GROUP_CONCAT()` | 将多行字段名和注释合并为一行字符串 |
| `COLUMN_NAME` | 列名 |
| `COLUMN_COMMENT` | 列注释 |
| `CONCAT()` | 拼接生成完整的 SELECT 语句 |

使用时将 `替换表名` 替换为实际的表名即可。
