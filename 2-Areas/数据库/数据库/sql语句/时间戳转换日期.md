# SQL 时间戳转换日期

使用 `FROM_UNIXTIME()` 函数将 Unix 时间戳转换为日期时间格式。

## 基本转换

```sql
SELECT FROM_UNIXTIME(time) AS 时间戳转换日期时间
FROM your_table;
```

转换效果：

| 转换前（时间戳） | 转换后（日期时间） |
| --- | --- |
| `1680856849` | `2023-04-07 08:40:49.000000` |
| `1680858635` | `2023-04-07 09:10:35.000000` |
| `1680922785` | `2023-04-08 02:59:45.000000` |

## 去掉小数点格式化

```sql
DATE_FORMAT(FROM_UNIXTIME(time), '%Y-%m-%d %H:%i:%s')
```

## 说明

| 函数 | 说明 |
| --- | --- |
| `FROM_UNIXTIME()` | 将 Unix 时间戳转为 `datetime` |
| `DATE_FORMAT()` | 格式化日期，`%Y-%m-%d %H:%i:%s` 精确到秒 |
