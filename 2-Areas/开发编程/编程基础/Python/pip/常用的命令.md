# 常用的命令

| 命令 | 说明 |
| --- | --- |
| pip3 install --upgrade pip | 升级pip工具自身到最新版本 |
| pip3 install -r requirements.txt | 从requirements.txt文件中安装所有依赖包 |
| pip3 install --index-url <url> | 使用指定的索引URL安装Python包 |
| pip3 install --trusted-host <host> <package> | 安装Python包时信任指定的主机 |
| pip3 install --no-index --find-links <directory> | 从指定目录中安装Python包而不使用索引 |
| pip3 install --upgrade --force-reinstall <package> | 强制重新安装指定的Python包 |
| pip3 freeze > requirements.txt | 将当前环境中安装的所有Python包及其版本号保存到requirements.txt文件中 |
| pip3 download <package> | 下载指定的Python包但不安装 |
| pip3 uninstall --yes <package> | 卸载指定的Python包并跳过确认提示 |

| pip3 list | 列出当前环境中安装的所有Python包及其版本号 |
| pip3 show <package> | 显示指定Python包的详细信息 |
| pip3 search <package> | 搜索指定Python包 |
| pip3 install --upgrade <package> | 升级指定的Python包到最新版本 |
| pip3 install --user <package> | 安装Python包到用户目录 |
| pip3 install --proxy <proxy> | 使用指定的代理服务器安装Python包 |
| pip3 install --no-cache-dir <package> | 禁止使用缓存目录安装Python包 |
