# 集合（set）

集合是一个无序的不重复元素序列，用{}表示

```python
set1=set([1,2,3,4,5,6,6,6,6,6])#列表转换成集合
set2=set((1,2,3,4,5,6,6,6,6,6))#元组转换成集合
set3=set(range(1,7)) #range对象转换成集合
set4={x for i in range(1,10)}  #集合推导式
set5=set()#创建一个新集合,不能使用{},因为它是创建一个字典
print(set1)
print(set2)
print(set3)
print(set4)

{1, 2, 3, 4, 5, 6}
{1, 2, 3, 4, 5, 6}
{1, 2, 3, 4, 5, 6}
{1, 2, 3, 4, 5, 6, 7, 8, 9}
"""
```
