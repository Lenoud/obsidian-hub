# 几个最常用的字符串处理的方法

| 方法及使用 | |
| --- | --- |
| str.lower()或str.upper
字母大小写转换 | 返回字符串的副本，全部字符 小写/大写
"AbCdEfgH".lower() 结果为"abcdefgh" |
| str.split(sep=None)
分隔字符 | 返回一个列表，由str根据sep被分隔的部分组成
"A,B,C".split(",")结果为['A','B','C'] |
| str.count(sub)
计算字符出现次数 | 返回子串sub在str中出现的次数
"an apple a day".count("a")结果为4 |
| str.replace(old,new)
替换旧字符串变为新字符串 | 返回字符串的str副本，所有old子串被替换为new
"python".replace("n","n123.io") 结果为"python123.io" |
| str.center(width[,fillchar])
使字符串居中，并且未满足宽度时填充字符 | "python".center(10,"=")结果为
'==python==' |
| str.strip(chars)
去掉列出的字符 | 从str中去掉在其左侧和右侧chars中列出的字符
"= python=".strip(" =np")结果为 (去掉了空格，n，p)
"ytho" |
| str.join(iter) | 在iter变量除最后元素外每个元素后增加一个str
",".join("12345")结果为
"1,2,3,4,5" |
| str.format | "{:-^10.2f}".format(a)
填充用 - 居中对齐 宽度为10字符 四舍五入到2位小数 浮点型 |
| list=[1,2,3,4]
list.insert(1,3)
在下标1插入一个3 | 在下标1插入一个3
list=[1,3,2,3,4] |
| s=data.find("PID:")
#获取字符串"PID:"的坐标位置
 | |
