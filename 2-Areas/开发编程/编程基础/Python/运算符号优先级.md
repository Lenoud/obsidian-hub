# Python 运算符优先级

Python 运算符从上到下优先级依次降低,**同级别按从左到右**顺序运算。括号 `()` 可改变优先级。

## 优先级表(高 → 低)

| 优先级 | 运算符 | 说明 | 示例 |
| --- | --- | --- | --- |
| 1(最高) | `**` | 指数 | `2 ** 3 == 8` |
| 2 | `~x` `+x` `-x` | 按位取反、正负号 | `~5 == -6` |
| 3 | `*` `/` `%` `//` | 乘、除、模、整除 | `7 // 2 == 3` |
| 4 | `+` `-` | 加、减 | — |
| 5 | `<<` `>>` | 按位左移、右移 | `1 << 3 == 8` |
| 6 | `&` | 按位与 | `5 & 3 == 1` |
| 7 | `^` | 按位异或 | `5 ^ 3 == 6` |
| 8 | `\|` | 按位或 | `5 \| 3 == 7` |
| 9 | `<` `<=` `>` `>=` `!=` `==` | 比较(链式) | `1 < x < 10` |
| 10 | `not x` | 逻辑非 | `not True == False` |
| 11 | `and` | 逻辑与(短路) | `True and False` |
| 12 | `or` | 逻辑或(短路) | `True or False` |
| 13(最低) | `if-else` `lambda` | 条件表达式、lambda | `x if cond else y` |

## 易错点

### 1. 指数 vs 负号

```python
-2 ** 2      # 等于 -(2**2) == -4,不是 (-2)**2
(-2) ** 2   # == 4
2 ** -1     # == 0.5(指数优先级高于负号)
```

### 2. 短路求值

```python
# and / or 是短路求值,可用来设置默认值
name = user_input or "default"

# 等价的三元表达式
name = user_input if user_input else "default"
```

### 3. 比较运算符链式

```python
# Python 支持链式比较,其他语言不支持
1 < x < 10          # 等价于 1 < x and x < 10
x == y == z         # 等价于 x == y and y == z
```

### 4. 位运算优先级陷阱

```python
# 位运算符的优先级低于比较运算符!这是反直觉的
x = 5
x & 1 == 1          # 等价于 x & (1 == 1),即 x & True
                    # 不是 (x & 1) == 1

# 正确写法
(x & 1) == 1
```

### 5. 赋值不是表达式(Python 3.8 以前)

```python
# Python 3.8 以前:赋值语句不能用在表达式里
# if x = func():   # 语法错误

# Python 3.8+:海象运算符 :=
if (n := len(data)) > 10:
    print(f"list too long: {n}")
```

## 实用规则

1. **拿不准就加括号**:可读性 > 记优先级表
2. 只有 `**` 的右结合需要注意:`2 ** 3 ** 2 == 2 ** (3 ** 2) == 512`
3. 自定义类的运算符优先级由 Python 固定,不能在类内修改
