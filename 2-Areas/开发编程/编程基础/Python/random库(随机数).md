# Python random 随机数库

Python 标准库 `random` 用于生成伪随机数，采用梅森旋转算法。

## 基本随机数函数

| 函数 | 描述 |
| --- | --- |
| `seed(a=None)` | 初始化随机数种子，默认为当前系统时间 |
| `random()` | 生成 `[0.0, 1.0)` 之间的随机小数 |

## 扩展随机数函数

| 函数 | 描述 |
| --- | --- |
| `randint(a, b)` | 生成 `[a, b]` 之间的整数 |
| `randrange(m, n[, k])` | 生成 `[m, n)` 之间以 k 为步长的随机整数 |
| `getrandbits(k)` | 生成 k 比特长的随机整数 |
| `uniform(a, b)` | 生成 `[a, b]` 之间的随机小数 |
| `choice(seq)` | 从序列中随机选取一个元素 |
| `shuffle(seq)` | 将序列元素随机排列 |
