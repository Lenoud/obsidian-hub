# conda 下载地址

conda / miniconda / anaconda 的官方与国内镜像下载地址。

## Miniconda(推荐,体积小)

只包含 conda + Python,按需安装包。

| 来源 | 链接 |
| --- | --- |
| 官方 | <https://docs.conda.io/en/latest/miniconda.html> |
| 清华镜像(国内推荐) | <https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/> |
| 中科大镜像 | <https://mirrors.ustc.edu.cn/anaconda/miniconda/> |
| 阿里云镜像 | <https://mirrors.aliyun.com/anaconda/miniconda/> |

### Linux 安装示例

```bash
# 下载(清华源)
wget https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/Miniconda3-latest-Linux-x86_64.sh

# 校验(可选)
sha256sum Miniconda3-latest-Linux-x86_64.sh

# 静默安装到 ~/miniconda3
bash Miniconda3-latest-Linux-x86_64.sh -b -p ~/miniconda3

# 初始化 shell(写入 ~/.bashrc / ~/.zshrc)
~/miniconda3/bin/conda init bash
source ~/.bashrc
```

## Anaconda(完整版,体积大)

预装数据科学常用包(numpy / pandas / jupyter / scipy 等),约 3GB。

| 来源 | 链接 |
| --- | --- |
| 官方 | <https://www.anaconda.com/download> |
| 清华镜像 | <https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/> |

## miniforge(社区版,默认 conda-forge)

miniforge 是社区维护的 miniconda 替代品,默认使用 conda-forge 渠道,适合 M1/M2 Mac 或需要最新版包的场景。

- GitHub Releases:<https://github.com/conda-forge/miniforge/releases>

## 国内源配置

安装后建议立刻切换到国内 conda 源:

```bash
# ~/.condarc
channels:
  - defaults
show_channel_urls: true
default_channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/r
custom_channels:
  conda-forge: https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud
```
