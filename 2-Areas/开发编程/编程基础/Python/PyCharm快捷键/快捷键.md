# PyCharm 快捷键

PyCharm 编辑器常用快捷键速查。

## 编辑文本
| **功能** | **快捷键** |
| :--- | :--- |
| 复制 | Ctrl + C |
| 剪切 | Ctrl + X |
| 粘贴 | Ctrl + V |
| 撤销 | Ctrl + Z |
| 重做 | Ctrl + Y 或 Ctrl + Shift + Z |
| 选中全部文本 | Ctrl + A |
| 查找 | Ctrl + F |
| 替换 | Ctrl + R |
| 查找下一个 | F3 |
| 查找上一个 | Shift + F3 |
| 跳转到行首 | Home |
| 跳转到行尾 | End |
| 跳转到指定行 | Ctrl + G |
| 注释/取消注释 | Ctrl + / |

## 导航和文件操作
| **功能** | **快捷键** |
| :--- | :--- |
| 打开文件 | Ctrl + O |
| 保存文件 | Ctrl + S |
| 关闭文件 | Ctrl + F4 |
| 新建文件 | Ctrl + N |
| 保存所有文件 | Ctrl + Shift + S |
| 切换窗口 | Alt + Tab |
| 切换项目视图 | Alt + 1 |
| 打开最近文件 | Ctrl + E |
| 关闭当前标签页 | Ctrl + F4 |
| 打开终端 | Alt + F12 |
| 打开设置 | Ctrl + Alt + S |

## 代码编辑和调试
| **功能** | **快捷键** |
| :--- | :--- |
| 缩进 | Tab 或 Shift + Tab |
| 提示补全 | Ctrl + Space |
| 显示参数信息 | Ctrl + P |
| 重构代码 | Ctrl + Alt + Shift + T |
| 运行程序 | Shift + F10 |
| 调试程序 | Shift + F9 |
| 添加断点 | Ctrl + F8 |
| 单步执行 | F8 |
| 进入方法/函数 | F7 |
| 跳出方法/函数 | Shift + F7 |
| 快速跳转到定义 | Ctrl + B |
