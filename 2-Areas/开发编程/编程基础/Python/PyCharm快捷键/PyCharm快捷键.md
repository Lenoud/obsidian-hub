# PyCharm快捷键

| **快捷键** | **操作描述** |
| --- | --- |
| **Ctrl** + **S** (Windows/Linux) or **Command** + **S** (macOS) | 保存当前文件 |
| **Ctrl** + **Z** (Windows/Linux) or **Command** + **Z** (macOS) | 撤销上一步操作 |
| **Ctrl** + **Y** (Windows/Linux) or **Command** + **Y** (macOS) | 重做上一步操作 |
| **Ctrl** + **C** (Windows/Linux) or **Command** + **C** (macOS) | 复制所选内容 |
| **Ctrl** + **X** (Windows/Linux) or **Command** + **X** (macOS) | 剪切所选内容 |
| **Ctrl** + **V** (Windows/Linux) or **Command** + **V** (macOS) | 粘贴剪贴板内容 |
| **Ctrl** + **F** (Windows/Linux) or **Command** + **F** (macOS) | 查找文本 |
| **Ctrl** + **R** (Windows/Linux) or **Command** + **R** (macOS) | 替换文本 |
| **Ctrl** + **D** (Windows/Linux) or **Command** + **D** (macOS) | 复制当前行或所选内容并插入下一行 |
| **Ctrl** + **/** (Windows/Linux) or **Command** + **/** (macOS) | 注释或取消注释当前行或所选内容 |
| **Ctrl** + **Shift** + **Z** (Windows/Linux) or **Command** + **Shift** + **Z** (macOS) | 重做上一步操作 |
| **Ctrl** + **E** (Windows/Linux) or **Command** + **E** (macOS) | 打开最近的文件 |
| **Ctrl** + **N** (Windows/Linux) or **Command** + **N** (macOS) | 快速打开文件或类 |
| **Ctrl** + **B** (Windows/Linux) or **Command** + **B** (macOS) | 转到定义（跳转到变量、方法或类的定义处） |
| **Ctrl** + **F12** (Windows/Linux) or **Command** + **F12** (macOS) | 显示当前文件的结构（方法、变量等） |
| **Alt** + **Enter** | 显示上下文相关的操作和快速修复选项 |
| **Ctrl** + **Space** | 启用代码自动完成 |
| **Ctrl** + **Alt** + **L** (Windows/Linux) or **Command** + **Option** + **L** (macOS) | 格式化代码 |
| **Ctrl** + **W** (Windows/Linux) or **Command** + **W** (macOS) | 逐渐增加所选内容的范围 |
| **Tab** | 缩进已选择的代码块。 |
| **Shift** + **Tab** | 将已选中的代码块向左（减少缩进） |
| **Ctrl** + **Alt** + **L** | 整理代码 |

| **快捷键** | **操作描述** |
| :--- | :--- |
| **Ctrl** + **Shift** + **F** (Windows/Linux) or **Command** + **Shift** + **F** (macOS) | 在项目中执行全局查找和替换 |
| **Ctrl** + **G** (Windows/Linux) or **Command** + **G** (macOS) | 跳转到指定的行号 |
| **Ctrl** + **F3** (Windows/Linux) or **Command** + **F3** (macOS) | 高亮选中的文本，然后按快捷键，将所有匹配项都高亮 |
| **Alt** + **Shift** + **C** | 在版本控制系统中提交代码 |
| **Ctrl** + **Alt** + **S** (Windows/Linux) or **Command** + **,** (macOS) | 打开 PyCharm 设置 |
| **Ctrl** + **Shift** + **A** (Windows/Linux) or **Command** + **Shift** + **A** (macOS) | 查找并执行操作、设置等 |
| **Ctrl** + **Alt** + **I** (Windows/Linux) or **Option** + **Command** + **I** (macOS) | 快速生成代码（自动生成 getter、setter、构造函数等） |
| **Ctrl** + **P** (Windows/Linux) or **Command** + **P** (macOS) | 显示函数参数提示 |
| **Ctrl** + **Alt** + **T** (Windows/Linux) or **Control** + **Option** + **T** (macOS) | 包围代码块（例如，包围选择的代码块以创建 **if**、**for**、**try** 等块） |
| **Ctrl** + **J** (Windows/Linux) or **Command** + **J** (macOS) | 显示代码模板 |
| **Ctrl** + **U** (Windows/Linux) or **Command** + **U** (macOS) | 跳转到父方法/函数 |
| **Alt** + **Insert** | 自动生成代码，如生成构造函数、getter 和 setter 等 |
| **Ctrl** + **H** (Windows/Linux) or **Command** + **H** (macOS) | 显示类层次结构 |
| **Ctrl** + **Shift** + **Enter** (Windows/Linux) or **Shift** + **Command** + **Enter** (macOS) | 自动补全当前行并开始新行 |
| **Ctrl** + **Shift** + **F10** (Windows/Linux) or **Control** + **Shift** + **R** (macOS) | 运行当前配置 |
| **Ctrl** + **Shift** + **F9** (Windows/Linux) or **Command** + **Shift** + **R** (macOS) | 构建项目 |
| **Ctrl** + **Alt** + **T** (Windows/Linux) or **Control** + **Option** + **T** (macOS) | 包围代码块（例如，包围选择的代码块以创建 **if**、**for**、**try** 等块） |
| **Ctrl** + **Shift** + **/** (Windows/Linux) or **Command** + **/** (macOS) | 注释或取消注释代码块 |
