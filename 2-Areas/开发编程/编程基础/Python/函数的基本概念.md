# Python 函数基本概念

Python 函数的定义、参数、作用域和 lambda 函数。

## 函数语法

```python
def <函数名>(<参数(0个或多个)>):
    <函数体>
    return <返回值>
```

## 局部变量与全局变量

- 局部变量在函数执行完成后会被释放
- 局部变量与全局变量互不影响（即使同名）
- `global` 保留字可以在函数中声明使用全局变量
- 如果函数内部未创建组合数据类型（如列表），修改它会影响全局变量

```python
# global 使用全局变量
a, b = 10, 100
def num(a):
    global b  # 使用全局变量 b=100
    return b
```

## lambda 函数

```python
f = lambda x, y: x + y
f(10, 15)  # 25

f = lambda: "nihao"
print(f())  # nihao
```
