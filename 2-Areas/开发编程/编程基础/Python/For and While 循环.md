# Python for 和 while 循环

Python 中 `for` 循环和 `while` 循环的用法与区别。

## 区别

| 特点 | for | while |
| --- | --- | --- |
| 适用场景 | 已知循环次数 | 未知循环次数 |
| 遍历 | 常用于遍历可迭代对象 | 很少用于遍历 |
| 停止条件 | 序列穷尽时停止 | 条件不成立时停止 |

## 循环控制关键字

### break

跳出并结束当前整个循环，执行循环后的语句。一个 `break` 只跳出一层循环。

### continue

结束本次循环，继续后续次数的循环。

## 循环与 else

循环可以与 `else` 搭配使用，在循环正常执行完毕后 `else` 会被执行。如果循环中有 `break`，一旦触发 `break`，`else` 不会被执行。
