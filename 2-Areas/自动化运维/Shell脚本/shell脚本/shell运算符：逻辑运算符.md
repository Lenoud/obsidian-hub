# Shell 运算符：逻辑运算符

Shell 逻辑运算符（&&、||）的使用方法。

本文介绍 Shell 中的逻辑运算符（&&、||、!），以及它们与布尔运算符的区别：&& 和 || 必须放在 [[ ]] 或 (()) 中使用。

> 功能与布尔运算符很相似
>
> 逻辑运算符与布尔运算符的区别
>
> 布尔运算符： ! -o -a   必须由[ ] 或者 test执行
>
> 逻辑运算符：&& ||  !
>
> &&和 || 的运算符必须放在 [[ ]] 或 (())使用
>
> ! 可以用在[ ]，[[ ]] 中，不可以在 (()) 使用

## 介绍
| 运算符 | 说明 | 用法 |
| --- | --- | --- |
| && | 逻辑的and | [[  1 > 2 &&  1==1 ]] |
| \|\| | 逻辑的or | [[  1 > 2  \|\|  1==1 ]] |
| ! | 逻辑非 | [[  !  1==1  ]] |

> 使用&&和 || 的运算符必须放在 [[ ]] 或 (()) 中才有效，否则报错
>
> ! 可以用在[ ]，[[ ]] 中，不可以在 (()) 使用
