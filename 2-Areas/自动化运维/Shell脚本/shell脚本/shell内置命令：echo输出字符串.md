# Shell 内置命令：echo 输出字符串

Shell 内置命令 echo 用于输出字符串。

本文介绍 Shell 中 echo 命令的用法，包括默认输出、不换行输出（-n）和解析转义字符（-e）的功能。

能够echo输出不换行的字符串，能够使用echo输出转义字符。

## echo命令介绍
echo是一个shell内置命令，用于在终端输出字符串，并在最后默认加上换行符

## echo输出语法
默认输出换行语法

```bash
echo 字符串
```

### 输出不换行语法 -n参数
```bash
echo -n 字符串
```

### 解析转义字符 -e参数
-e用于解析转义字符

```bash
echo -e "hello\nworld"
hello
world
```

| 转义字符 | 描述 |
| --- | --- |
| `\\` | 反斜杠 |
| `\'` | 单引号 |
| `\"` | 双引号 |
| `\n` | 换行 |
| `\r` | 回车 |
| `\t` | 制表符（水平） |
| `\v` | 垂直制表符 |
| `\b` | 退格 |
| `\f` | 换页 |
| `\a` | 响铃 |
| `\0` | 空字符（Null） |
| `\c` | 清除结尾的换行 |
| `\uXXXX` | Unicode字符（16位编码） |
| `\UXXXXXXXX` | Unicode字符（32位编码） |
| `\oNNN` | 八进制值 |
| `\xNN` | 十六进制值 |
