# Shell 计算命令：(()) 命令详解

Shell 双括号 (()) 数学运算命令详解。

本文详细介绍 Shell 中双小括号 (( )) 命令的用法，用于整数数学运算表达式的执行，支持赋值、逻辑运算和多表达式计算。

## 括号内表达式必须是整数表达式
> 双小括号 (( )) ,用于进行数学运算表达式的执行，将数学运算表达式放在 (()) 和之间。
>
> 可以使用$获取 (()) 表达式命令的结果，这和使用$获取变量值是一样的

## 语法
```bash
(( 整数运算表达式 ))
```

## 示例
| 运算操作符/运算命令 | 介绍 |
| --- | --- |
| ((a=1+6))((b=a-2))((c=a+b)) | 这种写法可以在计算完成后给变量赋值。以 ((b=a-1)) 为例，即将a-1的运算结果赋值给变量b。注意，使用变量时不用加$前缀，(()) 会自动解析变量名。 |
| a=$((1+6))b=$((a-1))c=$((a+b)) | 通过 $ 获取运算结果赋值给变量 |
| ((a>7 && b==c)) | 可以直接进行逻辑运算 |
| echo $((a+10)) | 可以直接获取表达式的值 |
| ((a=1+3,b=a+10)) | 多表达式赋值，使用英文逗号分隔 |
