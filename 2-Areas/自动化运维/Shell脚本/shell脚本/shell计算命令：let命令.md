# Shell 计算命令：let 命令

Shell let 命令用于整数运算。

本文介绍 Shell 中 let 命令的用法，let 用于整数赋值计算，是最简洁的整数运算赋值命令，但不支持直接输出和条件判断。

> let命令和双小括号 (( )) 在数字计算方面功能一样.但是没有 (( )) 功能强大，let只能用于赋值计算，不能直接输出，不可以条件判断一起使用

## 示例
```bash
#!/bin/bash
# let 变量名=表达式
let a=1+6
let b=a-1
let c=a+b

echo "a=${a},b=${b},c=${c}"

#多个表达式赋值
let a=1+6  b=a-1 c=a+b

echo "a=${a},b=${b},c=${c}"
```

## 作用
用于赋值，是最简洁的整数运算赋值命令
