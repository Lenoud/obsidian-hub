# 自定义全局变量

Shell 脚本中自定义全局变量（export）的方法。

### 父子shell环境介绍
有2个shell脚本文件A.sh B.sh

如果在A.sh脚本文件中执行了B.sh脚本文件，那么A.sh就是夫shell环境，b.sh就是子shell环境

### 自定义全局变量介绍
就是在当前脚本文件中定义全局变量，这个全局变量可以在当前shell环境与子shell环境中都可以使用

### 自定义全局变量语法
```shell
export var_name1 var_name2=value
```

### 例子
```shell
[root@rockylinux-docker shell]# cat A.sh
#!/bin/bash
export VAR4="nihao"
bash /root/shell/B.sh
[root@rockylinux-docker shell]# cat B.sh
#!/bin/bash
echo "var4=${VAR4}"

#这样B就可以读取到A中定义的全局变量
```
