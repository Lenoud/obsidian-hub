# Shell 脚本

Shell 脚本编程入门与进阶笔记的总目录,涵盖变量、运算符、流程控制、函数、计算命令、文本处理等核心知识点。

## 学习路径

```text
1. 入门   →  变量 / 运算符 / 输入输出
2. 进阶   →  流程控制(if/for/while/case/until/select) / 函数
3. 计算   →  expr / (()) / $[] / let / bc
4. 文本   →  grep / sed / awk(见 Linux 三剑客)
5. 实战   →  系统初始化 / 自动化部署 / 运维脚本
```
