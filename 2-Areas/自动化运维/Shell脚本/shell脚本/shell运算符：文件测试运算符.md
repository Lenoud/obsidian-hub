# shell运算符：文件测试运算符

Shell 文件测试运算符（-f、-d、-e 等）的使用方法。

### linux系统文件类型介绍
| 符号 |  文件类型   |
| --- | --- |
| - | 普通文件 |
| d | 目录文件 |
| l | 符号链接 |
| b | 块设备文件 |
| c | 字符设备文件 |
| p | 管道文件 |

### 文件测试运算符介绍
> 可以使用双方括号，也可以使用单方括号，推荐双
>

| 语法 | 含义 | 用例 |
| --- | --- | --- |
| -e file | 检测文件（目录）是否存在 | `file="/root/text.txt"` `[[ -e ${file} ]]` |
| -d file | dir，检查是否为目录 |   |
| -f file | file，检查是否为文件 | |
| -r file | read，可否可读 | |
| -w file | write，是否可写 | |
| -x file | execute，是否可执行 | |
| -s file | size，是否为空 | |
| -nt | new than | |
| -ot | old than | |

### 文件类型详细说明
#### 1. **普通文件（**`**-**`**）**
+ 存储用户数据，可能是文本、程序或图片等。
+ 扩展名（如 `.txt`, `.c`, `.jpg`）有助于区分用途。

#### 2. **目录文件（**`**d**`**）**
+ 记录文件名和对应的 inode 信息。
+ 提供层次化的文件系统组织。

#### 3. **符号链接（**`**l**`**）**
+ 类似于快捷方式，指向另一个文件或目录。
+ 可以通过 `ln -s` 创建符号链接。

#### 4. **块设备文件（**`**b**`**）**
+ 用于存储设备（如硬盘）。
+ 常位于 `/dev` 目录下，例如 `/dev/sda`。

#### 5. **字符设备文件（**`**c**`**）**
+ 提供按字符流处理的设备接口。
+ 常见例子：`/dev/tty`（终端设备）。

#### 6. **套接字文件（**`**s**`**）**
+ 用于进程间通信（IPC），支持网络连接。
+ 常见于 `/var/run` 目录下。

#### 7. **命名管道（**`**p**`**）**
+ 一种特殊的文件类型，用于进程间通信。
+ 使用 `mkfifo` 命令创建。

### 其他补充工具
+ `**file**`** 命令**：识别文件类型。例如：

```bash
file filename
```

+ `**stat**`** 命令**：查看文件详细信息。例如：

```bash
stat filename
```
