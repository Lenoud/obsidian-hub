# Shell 运算符：布尔运算符

Shell 布尔运算符（!、-a、-o）的使用方法。

本文介绍 Shell 中的布尔运算符（!、-o、-a），布尔运算符需要放在 [ ] 中或与 test 命令结合使用。

> 布尔运算符放在 [ ] 中，或者与test命令结合使用，一般情况下都是配合test使用

## 介绍
| 运算符 | 说明 | 用法 |
| --- | --- | --- |
| ! | 取反 | [ ! 1 \> 2 ]  返回0，本来是不成立，取反返回成立 |
| -o | or 或者，有一个表达式为true则返回0 | [  1 \> 2  -o 1 == 1 ]  有一个条件成立，返回0 |
| -a | and 并且，两个表达式都为true才返回0 | [  1 \> 2  -a 1 == 1 ]   需要所有条件成立，短路操作：(如果第一个表达式失败了，直接返回，不会去判断第二个表达式) |
