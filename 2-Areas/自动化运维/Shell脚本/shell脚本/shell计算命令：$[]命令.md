# Shell 计算命令：$[] 命令

Shell $[] 数学运算语法。

本文介绍 Shell 中 $[] 命令的用法，用于对单个整数表达式进行计算求值与输出，注意 $[] 中只能放置一个表达式且不能直接赋值给变量。

## 单个表达式的计算求值与输出
> 和(())、let命令类似，$[]也只能进行整数运算。但是只能对单个表达式的计算求值与输出

不能在 $[] 中直接赋值给变量！

## 示例
```bash
#[]中只能放置一个表达式

[root@ansible shell]# a=$[1+2]
[root@ansible shell]# b=$[a+3]
[root@ansible shell]# echo "$[a+b]"
9
```
