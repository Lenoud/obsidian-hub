# Shell 加载流程原理介绍

Shell 脚本加载流程原理（登录 shell 与非登录 shell）。

本文介绍 Shell 的分类概念（交互式/非交互式、登录/非登录）以及环境变量加载初始化流程。

## 交互式与非交互式shell

### 交互式shell
是需要用户参与互动的shell环境，需要用户输入shell命令，提交到内核立马给出响应的就是交互式shell

### 非交互式shell
只执行命令，不需要用户的参与

## 登录shell与非登录shell

### 登录shell
要以用户名与密码登录到系统。默认采用登录shell环境。

### 非登录shell
不使用用户名与密码进入Linux系统的shell环境。
