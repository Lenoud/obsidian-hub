# 字符串变量：拼接

Shell 字符串拼接方法。

### 无符号拼接
```shell
[root@rockylinux-docker shell]# var1=abc
[root@rockylinux-docker shell]# var2=def
[root@rockylinux-docker shell]# var3=${var1}${var2}

[root@rockylinux-docker shell]# echo $var3
abcdef

```

### 双引号拼接
```shell
[root@rockylinux-docker shell]# var3="${var1}&&##${var2}"
[root@rockylinux-docker shell]# echo $var3
abc&&##def

```

### 混合拼接
```shell
[root@rockylinux-docker shell]# var3=${var1}'$12321412'${var2}
[root@rockylinux-docker shell]# echo $var3
abc$12321412def

[root@rockylinux-docker shell]# var3=${var1}"12321412"${var2}

#也不能使用空格拼接
#如果想使用空格拼接只能使用echo

[root@rockylinux-docker shell]# echo nihao liubiao llasd asdf ${var1}
nihao liubiao llasd asdf abc

```
