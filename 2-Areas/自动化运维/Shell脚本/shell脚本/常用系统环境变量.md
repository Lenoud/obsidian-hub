# 常用系统环境变量

本文列出 Shell 中常用的系统环境变量及其含义，包括 HOME、PATH、USER、SHELL、PWD 等。

## 常用系统环境变量列表

| 变量名 | 含义 |
| --- | --- |
| $HOME | 当前用户的主目录 |
| $PATH | 可执行命令的搜索路径 |
| $USER | 当前用户名 |
| $SHELL | 当前使用的 Shell 类型 |
| $PWD | 当前工作目录 |
| $HOSTNAME | 主机名 |
| $LANG | 系统语言环境 |
| $UID | 当前用户 ID |
| $RANDOM | 生成一个随机数 |
| $IFS | 内部字段分隔符，默认为空格、制表符、换行符 |
