# 自动化工具介绍

主流自动化运维工具（Ansible、SaltStack、Puppet 等）对比介绍。

本文对比介绍三款主流自动化运维工具（Ansible、SaltStack、Puppet）的功能特点和适用场景。

| **工具** | 功能 | 描述 |
| --- | --- | --- |
| **Ansible** | 配置管理、应用部署、任务自动化 | 无代理架构、易于学习和使用、模块化设计、强大的社区支持 |
| **SaltStack** | 配置管理、远程执行、事件驱动的自动化 | 需要agent端配合，配置部署速度快，高速并行操作、支持实时事件响应、模块化架构 |
| **Puppet** | 配置管理、系统状态管理 | 老牌配置自动化工具，需要agent端配合，强大的状态管理、适合大型基础设施、基于声明式模型 |
