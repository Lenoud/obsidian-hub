# Ansible 基本操作

Ansible 基本操作命令速查。

本文列出 Ansible 的常用基础操作命令，包括运行 Playbook、主机连通性测试、命令执行、角色初始化、文件加密解密、模块文档查看和主机清单展示。

| 命令 | 功能 |
| --- | --- |
| ansible-playbook playbook.yml | 运行指定的Playbook |
| ansible \<hosts\> -m ping | 对指定主机执行ping测试 |
| ansible \<hosts\> -a "command" | 在指定主机上执行特定命令 |
| ansible-galaxy init role_name | 初始化一个新的Ansible角色 |
| ansible-vault encrypt file | 对文件进行加密 |
| ansible-vault decrypt file | 对文件进行解密 |
| ansible-doc module_name | 查看特定模块的文档 |
| ansible-inventory --graph | 以图表形式显示主机清单 |
