# 修改前端跳转代码

修改前端跳转代码相关技术笔记。

```html
\\cat /www/index.html

<!DOCTYPE html>
<html>
  <head>
    <meta http-equiv="Cache-Control" content="no-cache, no-store, must-revalidate" />
    <meta http-equiv="Pragma" content="no-cache" />
    <meta http-equiv="Expires" content="0" />
    <meta http-equiv="refresh" content="3; URL=cgi-bin/luci/" />
    <style>
      body {
        background: #222;
        color: white;
        font-family: 'Arial', sans-serif;
        display: flex;
        justify-content: center;
        align-items: center;
        height: 100vh;
        margin: 0;
        padding: 0;
      }

      .redirect-message {
        text-align: center;
        animation: fadein 2s;
      }

      @keyframes fadein {
        from { opacity: 0; }
        to   { opacity: 1; }
      }
    </style>
  </head>
  <body>
    <div class="redirect-message">
      <h1>Redirecting...</h1>
      <p>Please wait while we redirect you to the Lua Configuration Interface.</p>
    </div>
  </body>
</html>
```
