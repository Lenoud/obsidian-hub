# 简洁监控工具

Ward 是一款轻量级的服务器监控工具,提供直观的系统资源(CPU / 内存 / 磁盘 / 网络)实时仪表盘。

```yaml
version: '3.3'
services:
    ward:
        restart: unless-stopped
        container_name: ward
        ports:
            - '4000:4000'
        environment:
            - WARD_PORT=4000
            - WARD_THEME=dark
            - WARD_NAME=leons-server
            - WARD_FOG=false
            - WARD_BACKGROUND=#3c3c3c
        privileged: true
        volumes:
          - /etc/localtime:/etc/localtime
        image: antonyleons/ward:2.5.2
```
