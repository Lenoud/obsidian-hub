# 简洁监控工具

简洁监控工具相关技术笔记。

```bash
version: '3.3'
services:
    ward:
        restart: unless-stopped
        container_name: ward
        ports:
            - '4000:4000'
        environment:
            - WARD_PORT=4000
            - WARD_THEME=dark
            - WARD_NAME=leons-server
            - WARD_FOG=false
            - WARD_BACKGROUND=#3c3c3c
        privileged: true
        volumes:
          - /etc/localtime:/etc/localtime
        image: antonyleons/ward:2.5.2

```
