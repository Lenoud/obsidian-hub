# Windows Shutdown 命令详解

Windows `shutdown` 命令支持通过命令行控制计算机的关机、重启、注销等操作。

---

### **1. 基础操作参数**
+ `/s`
立即关闭计算机。
**示例**：`shutdown /s`
**用途**：直接关机，默认延迟 30 秒（可通过 `/t` 修改）。
+ `/r`
立即重启计算机。
**示例**：`shutdown /r`
**用途**：重启系统，适用于更新后或故障修复。
+ `/l`
注销当前用户。
**示例**：`shutdown /l`
**用途**：快速切换用户或结束当前会话。
+ `/h`
进入休眠模式（需硬件支持）。
**示例**：`shutdown /h`
**用途**：低功耗保存当前状态，快速恢复工作。

---

### **2. 定时与延迟参数**
+ `/t <秒数>`
设置关机/重启前的延迟时间（单位：秒，范围 0-315360000）。
**示例**：`shutdown /s /t 3600`（1 小时后关机）
**注意**：默认 30 秒，若未指定会弹窗提醒。
+ `/a`
取消计划的关机或重启。
**示例**：`shutdown /a`
**适用场景**：终止已通过 `/t` 设置的倒计时任务。

---

### **3. 高级功能参数**
+ `/f`
强制关闭所有运行中的程序，不提示用户保存。
**示例**：`shutdown /r /f`
**注意**：可能导致数据丢失，慎用！
+ `/m \\计算机名`
远程关闭或重启其他计算机（需权限）。
**示例**：`shutdown /s /m \\192.168.1.100`
**要求**：目标计算机开启 `Remote Registry` 服务，且用户有管理员权限。
+ `/hybrid`
执行混合关机（Win8+ 支持），结合快速启动和完全关机。
**示例**：`shutdown /s /hybrid`
**特点**：下次开机速度更快，但会保存系统状态到硬盘。
+ `/e`
记录关机的详细原因（需结合事件查看器查看）。
**示例**：`shutdown /s /e /c "系统维护"`

---

### **4. 组合使用示例**
+ **立即强制重启**：
`shutdown /r /f`
+ **5 分钟后远程关机**：
`shutdown /s /m \\OfficePC /t 300`
+ **取消计划关机**：
`shutdown /a`

---

### **注意事项**
1. **权限要求**：部分操作（如远程关机）需以管理员身份运行命令提示符（CMD 或 PowerShell）。
2. **参数冲突**：某些参数不可同时使用（如 `/s` 和 `/r`）。
3. **日志查看**：使用 `/e` 参数后，可在 **事件查看器 → Windows 日志 → 系统** 中查看关机原因。

掌握这些参数可以灵活管理系统开关机流程，尤其适用于批量维护或自动化脚本场景。
