# Excel 判断单元格值填充颜色

使用 openpyxl 库根据 Excel 单元格的值自动填充背景颜色。

## 代码

```python
from openpyxl.styles import PatternFill
import openpyxl

# 打开 Excel 文件
workbook = openpyxl.load_workbook(r'F:\桌面\附表 - 副本.xlsx')
ws = workbook.worksheets[0]

# 设置绿色填充样式
fill = PatternFill(start_color='FF00FF00', end_color='FF00FF00', fill_type='solid')

# 匹配列表
search_list = ['value1', 'value2', 'value3']

# 遍历 B 列中的每个单元格
for cell in ws['B']:
    if cell.value in search_list:
        cell.fill = fill

# 保存修改后的 Excel 文件
workbook.save(r'F:\桌面\附表 - 副本.xlsx')
```

## 说明

- `PatternFill(start_color='FFFF0000', ...)` — 红色填充
- `PatternFill(start_color='FF00FF00', ...)` — 绿色填充
- `fill_type='solid'` — 实心填充
