# Excel 插入数据(openpyxl)

使用 Python 的 openpyxl 库向 Excel(.xlsx)文件写入数据,适合自动化报表、数据导出。

## 安装

```bash
pip install openpyxl
```

## 基本用法

```python
import openpyxl

# 新建工作簿
wb = openpyxl.Workbook()
ws = wb.active              # 默认工作表 "Sheet"
ws.title = "销售数据"

# 1. 单元格逐个写入
ws['A1'] = "月份"
ws['B1'] = "销售额"
ws['A2'] = "2024-01"
ws['B2'] = 12500

# 2. 一次写入一行(append)
ws.append(["2024-02", 18300])
ws.append(["2024-03", 21600])

# 3. 批量写入(从二维列表)
data = [
    ["2024-04", 19500],
    ["2024-05", 24800],
    ["2024-06", 31200],
]
for row in data:
    ws.append(row)

# 保存
wb.save("sales.xlsx")
```

## 公式与样式

```python
# 公式
ws['B7'] = "=SUM(B2:B6)"
ws['B8'] = "=AVERAGE(B2:B6)"

# 单元格样式
from openpyxl.styles import Font, PatternFill, Alignment

header_font = Font(bold=True, color="FFFFFF", size=12)
header_fill = PatternFill("solid", fgColor="4472C4")

for col in range(1, 3):
    cell = ws.cell(row=1, column=col)
    cell.font = header_font
    cell.fill = header_fill
    cell.alignment = Alignment(horizontal="center")

# 列宽
ws.column_dimensions['A'].width = 12
ws.column_dimensions['B'].width = 15

# 数字格式
ws['B2'].number_format = '#,##0.00'
```

## 打开已有文件追加

```python
wb = openpyxl.load_workbook("existing.xlsx")
ws = wb["Sheet1"]

# 找到最后一行
max_row = ws.max_row
ws.append([f"new-{max_row+1}", "value"])     # 追加到末尾

wb.save("existing.xlsx")
```

## 处理大数据(只读模式)

```python
# read_only 模式:不加载整个文件到内存,适合读大表
wb = openpyxl.load_workbook("large.xlsx", read_only=True)
ws = wb.active
for row in ws.iter_rows(values_only=True):
    print(row)
```

## 常见问题

| 问题 | 解决 |
| --- | --- |
| 写入报 `PermissionError` | 文件被 Excel 打开,关闭后重试 |
| 中文乱码 | 检查文件编码,openpyxl 默认 utf-8 |
| 大文件速度慢 | 用 `write_only=True` 流式写,或换 pandas |
| 不支持 .xls 旧格式 | openpyxl 只支持 .xlsx,.xls 用 xlwt/xlrd |
| 数字被存成字符串 | 用 `cell.value = 123`(int)而不是 `"123"`(str) |
