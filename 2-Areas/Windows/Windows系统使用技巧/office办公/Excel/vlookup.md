# VLOOKUP 函数用法

Excel VLOOKUP 函数用于在表格中按列查找数据。

## 语法

```
=VLOOKUP(查找值, 表格范围, 列序号, 匹配方式)
```

## 示例

### 表 A（用户数据）

| 用户id(A列) | 余额 | 等级 |
| --- | --- | --- |
| 100 | 10023 | `=VLOOKUP(A2, 表B!A2:C5, 3, 0)` |
| 104 | 123 | |
| 101 | 8910 | |

### 表 B（等级对照）

| 用户id(A) | 账号 | 等级(C列) |
| --- | --- | --- |
| 100 | 131231242121 | 13 |
| 101 | 161232141233 | 16 |
| 102 | 231212333122 | 23 |
| 103 | 172141565757 | 17 |
| 104 | 901234214456 | 90 |

### 参数解析

- `A2`：表 A 的第二行用户 id，即 100
- `表B!A2:C5`：表 B 中三列五行的查找范围
- `3`：返回查找范围的第三列（等级）
- `0`：精确匹配（FALSE）
