# 用户和权限管理

用户和权限管理相关技术笔记。

## 一．实训过程
### 1. 增加学生组
**目标：** 创建一个名为`students`的组，GID号设置为5000。

**操作步骤：**

1. 打开终端并切换到`root`用户。
2. 使用以下命令创建组：

```bash
groupadd -g 5000 students
```

### 2. 创建用户`user1`
**目标：**

+ 创建用户`user1`，UID号为5000。
+ 设置密码为空。
+ 将其附属组设置为`students`组。

**操作步骤：**

1. 使用以下命令创建用户：

```bash
useradd -u 5000 -g students user1
```

2. 设置用户密码为空：

```bash
passwd -d user1
```

### 3. 创建用户`user2`
**目标：**

+ 创建用户`user2`，密码设置为`123456`。
+ 设置其附属组为`root`组和`students`组。
+ 主目录设置为`/user2`。

**操作步骤：**

1. 创建用户并指定主目录：

```bash
useradd -m -d /user2 -G root,students user2
```

2. 设置密码：

```bash
echo "123456" | passwd --stdin user2
```

### 4. 创建用户`user3`
**目标：**

+ 不建立宿主目录。
+ 禁用其登录系统的Shell。

**操作步骤：**

1. 使用以下命令创建用户：

```bash
useradd -M -s /sbin/nologin user3
```

### 5. 设置`user1`用户密码策略
**目标：** 强制`user1`在下次登录时更改密码。

**操作步骤：**

1. 使用以下命令：

```bash
chage -d 0 user1
```

### 6. 设置`user2`用户密码有效期
**目标：**

+ 要求`user2`每30天更改密码。
+ 密码在2024年10月10日过期。

**操作步骤：**

1. 设置密码更改周期：

```bash
chage -M 30 user2
```

2. 设置密码到期时间：

```bash
chage -E 2024-10-10 user2
```

### 7. 创建目录并设置权限
**目标：**

+ 在`/var/www`下创建目录`user1`。
+ 设置不同的访问权限。

**操作步骤：**

1. 创建目录并设置所有者：

```bash
mkdir -p /var/www/user1
chown user1:students /var/www/user1
```

2. 设置权限：

```bash
chmod 750 /var/www/user1
```

3. 确保目录权限满足以下要求：
    - 所有者`user1`具有读写执行权限：
权限`rwx` (7)。
    - 组`students`具有读执行权限：
权限`r-x` (5)。
    - 其他用户只读权限：
权限`r--` (4)。
