# vim快捷键

Vim 编辑器常用快捷键速查表。

| **快捷键** | **描述** |
| --- | --- |
| i | 在光标处插入 |
| I | 在当前行首插入 |
| a | 在光标后插入 |
| A | 在当前行尾插入 |
| o | 在当前行之后插入新行并进入插入模式 |
| O | 在当前行之前插入新行并进入插入模式 |
| hjkl | 左下上右（光标移动） |
| w | 向前跳到下一个单词开头 |
| b | 向后跳到上一个单词开头 |
| e | 跳到当前单词结尾 |
| gg | 跳到文件开头 |
| G | 跳到文件末尾 |
| dd | 删除当前行 |
| dw | 删除一个单词（从光标处删除到下一个单词开头） |
| D | 删除到行尾 |
| x | 删除当前字符 |
| u | 撤销上一步操作 |
| Ctrl + r | 重做上一步操作 |
| yy | 复制当前行 |
| p | 粘贴 |
| /pattern | 向前搜索指定的内容 |
| ?pattern | 向后搜索指定的内容 |
| :w | 保存当前文件 |
| :q | 退出 Vim |
| :q! | 强制退出 Vim（不保存修改） |
| :x | 保存并退出 Vim |
| :wq | 保存并退出 Vim |
| :set nu | 显示行号 |
| :set nonu | 隐藏行号 |
| :%s/pattern/replace/g | 替换指定内容 |
| :help | 打开帮助文档 |
