# vi&vim操作技巧

Vim（Vi IMproved）是 Linux 下最常用的全屏文本编辑器，几乎所有发行版都内置 vi，装上 vim-enhanced 后可获得语法高亮、多级撤销等增强功能。

## 三种工作模式

Vim 启动后默认进入 **命令模式（Normal mode）**，在该模式下按键被解释为命令而非字符输入。三种模式之间的切换关系如下：

| 模式 | 别名 | 作用 | 切换方式 |
| --- | --- | --- | --- |
| 命令模式 | Normal mode | 光标移动、复制粘贴、删除、搜索 | 启动默认；任意模式下按 `Esc` 回到此处 |
| 插入模式 | Insert mode | 像普通编辑器一样输入文本 | 命令模式下按 `i` / `a` / `o` 等进入 |
| 底线命令模式 | Command-line / Ex mode | 保存、退出、替换、设置选项 | 命令模式下输入 `:` / `/` / `?` 进入 |

> 记忆口诀：**Esc 回命令，`:` 进底线，`i/a/o` 进插入**。

## 进入插入模式

以下按键均用于从命令模式切换到插入模式，区别在于光标落点：

| 按键 | 作用 |
| --- | --- |
| `i` | 在光标**前**插入 |
| `a` | 在光标**后**插入 |
| `I` | 在当前行**行首**插入 |
| `A` | 在当前行**行末**插入 |
| `o` | 在当前行**下方**新开一行并进入插入 |
| `O` | 在当前行**上方**新开一行并进入插入 |
| `s` | 删除光标字符并进入插入 |
| `cc` / `S` | 清空当前行并进入插入 |

## 命令模式常用操作

### 光标移动

| 按键 | 作用 | 按键 | 作用 |
| --- | --- | --- | --- |
| `h` `j` `k` `l` | 左 下 上 右 | `w` / `b` | 下一/上一个单词首 |
| `0` | 行首 | `^` | 行首第一个非空字符 |
| `$` | 行末 | `gg` | 跳到文件首行 |
| `G` | 跳到文件末行 | `:N` 或 `NG` | 跳到第 N 行 |
| `Ctrl+f` | 下翻一页 | `Ctrl+b` | 上翻一页 |
| `{` / `}` | 上/下一个段落 | `%` | 匹配括号跳转 |

### 复制 / 粘贴 / 删除

| 按键 | 作用 |
| --- | --- |
| `yy` | 复制（yank）当前行 |
| `yw` | 复制一个单词 |
| `y$` | 复制到行末 |
| `dd` | 剪切当前行 |
| `dw` | 剪切一个单词 |
| `x` / `X` | 删除光标处/前一个字符 |
| `p` / `P` | 粘贴到光标后/前 |
| `u` | 撤销（undo） |
| `Ctrl+r` | 重做（redo） |
| `.` | 重复上一次修改 |

### 前置计数

绝大多数命令支持前置数字，表示重复次数。例如：

- `3dd` —— 删除 3 行
- `5j` —— 向下移动 5 行
- `2yy` —— 复制 2 行
- `d10G` —— 删除到第 10 行

### 搜索

| 按键 | 作用 |
| --- | --- |
| `/word` | 向下搜索 word |
| `?word` | 向上搜索 word |
| `n` / `N` | 重复/反向重复搜索 |
| `*` / `#` | 向下/向上搜索光标所在单词 |

## 底线命令模式

在命令模式下输入 `:` 进入，常用命令：

| 命令 | 作用 |
| --- | --- |
| `:w` | 保存 |
| `:q` | 退出 |
| `:wq` / `:x` | 保存并退出 |
| `:q!` | 强制退出不保存 |
| `:w file` | 另存为 file |
| `:e file` | 打开文件 file |
| `:set nu` / `:set nonu` | 显示/隐藏行号 |
| `:set ai` | 自动缩进 |
| `:set paste` | 粘贴模式（避免自动缩进错乱） |
| `:%s/old/new/g` | 全文替换 old 为 new |
| `:s/old/new/g` | 当前行替换 |
| `:1,10s/old/new/g` | 1-10 行替换 |
| `:!cmd` | 执行外部 shell 命令 |
| `:r !cmd` | 把命令输出插入到当前位置 |

### 替换标志

- `g` —— 替换行内所有匹配
- `i` —— 忽略大小写
- `c` —— 替换前确认
- `e` —— 不报错

## 可视化模式

| 按键 | 模式 | 作用 |
| --- | --- | --- |
| `v` | 字符可视化 | 按字符选择 |
| `V` | 行可视化 | 按整行选择 |
| `Ctrl+v` | 块可视化 | 矩形块选择，可批量注释 |

**批量注释示例**（多行行首加 `#`）：

1. `Ctrl+v` 进入块选择，选中所有要注释的行首列
2. `Shift+i` 输入 `#`，再按 `Esc`
3. 稍等片刻，选中的所有行都会被加上 `#`

## 简单的 .vimrc 配置示例

将以下内容写入 `~/.vimrc` 即可获得较友好的默认行为：

```vim
set nocompatible
set number
set relativenumber
set tabstop=4
set shiftwidth=4
set expandtab
set autoindent
set smartindent
set cursorline
set showmatch
set hlsearch
set incsearch
set ignorecase smartcase
set wildmenu
set encoding=utf-8
set fileencodings=utf-8,gbk,gb18030,big5
syntax on
```

| 选项 | 含义 |
| --- | --- |
| `expandtab` | 输入 Tab 时用空格替代 |
| `hlsearch` / `incsearch` | 高亮 + 实时搜索 |
| `ignorecase smartcase` | 全小写忽略大小写，含大写则精确 |
| `wildmenu` | 命令行 Tab 补全菜单 |
