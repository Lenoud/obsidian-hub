# 网络操作

Linux 网络配置与排障涉及三大命令家族：`net-tools`（已废弃）、`iproute2`（现代标准）和 `ethtool`（网卡物理层）。本文整理速查对照表，便于在 CentOS 7 等系统上日常运维。

## 三大命令家族对比

| 家族 | 代表命令 | 状态 | 说明 |
| --- | --- | --- | --- |
| **net-tools** | `ifconfig` `route` `netstat` `arp` | **已废弃** | 来源于 2001 年前的工具集，CentOS 7+ 已默认不安装 |
| **iproute2** | `ip` `ss` `tc` | **现代标准** | 替代 net-tools，功能更全、性能更好 |
| **ethtool** | `ethtool` | 物理层工具 | 查看/设置网卡速率、双工、驱动、offload 等硬件参数 |

新旧命令对照：

| 功能 | net-tools（旧） | iproute2（新） |
| --- | --- | --- |
| 查看 IP 地址 | `ifconfig` | `ip addr` / `ip a` |
| 查看网卡链路 | `ifconfig -a` | `ip link` / `ip l` |
| 查看路由表 | `route -n` | `ip route` / `ip r` |
| 查看 ARP 缓存 | `arp -n` | `ip neigh` / `ip n` |
| 查看网络连接 | `netstat -anp` | `ss -anp` |
| 配置 IP | `ifconfig eth0 10.0.0.1/24` | `ip addr add 10.0.0.1/24 dev eth0` |

## ip 命令速查

`ip` 是 iproute2 套件的核心命令，语法统一为 `ip [OPTIONS] OBJECT {COMMAND}`。

### 通用速查表

| 对象 | 命令 | 作用 |
| --- | --- | --- |
| addr | `ip addr` / `ip a` | 查看所有接口的 IP |
| addr | `ip addr show dev eth0` | 查看 eth0 的 IP |
| addr | `ip addr add 10.0.0.5/24 dev eth0` | 给 eth0 临时加 IP |
| addr | `ip addr del 10.0.0.5/24 dev eth0` | 删除指定 IP |
| link | `ip link` / `ip l` | 查看所有网卡链路状态 |
| link | `ip -s link` | 带收发包统计 |
| link | `ip link set eth0 up` / `down` | 启用 / 停用网卡 |
| link | `ip link set eth0 mtu 9000` | 设置 MTU（巨型帧） |
| route | `ip route` / `ip r` | 查看路由表 |
| route | `ip route add 192.168.2.0/24 via 10.0.0.1` | 加静态路由 |
| route | `ip route del 192.168.2.0/24` | 删除路由 |
| route | `ip route add default via 10.0.0.1` | 设置默认网关 |
| rule | `ip rule` | 查看策略路由规则 |
| rule | `ip rule add from 192.168.1.10 table 100` | 添加策略路由 |
| neigh | `ip neigh` / `ip n` | 查看 ARP / 邻居表 |
| neigh | `ip neigh flush all` | 清空邻居缓存 |

### 查看接口 IP

```bash
ip addr show dev eth0
```

输出示例：

```text
2: eth0: <BROADCAST,MULTICAST,UP,LOWER_UP> mtu 1500 qdisc mq state UP group default qlen 1000
    link/ether 52:54:00:12:34:56 brd ff:ff:ff:ff:ff:ff
    inet 10.0.0.10/24 brd 10.0.0.255 scope global eth0
       valid_lft forever preferred_lft forever
    inet6 fe80::5054:ff:fe12:3456/64 scope link
       valid_lft forever preferred_lft forever
```

### 查看路由

```bash
ip route
```

输出示例：

```text
default via 10.0.0.1 dev eth0
10.0.0.0/24 dev eth0 proto kernel scope link src 10.0.0.10
192.168.1.0/24 via 10.0.0.254 dev eth0
```

## ifconfig 速查（兼容老系统）

部分老旧系统或脚本仍依赖 ifconfig，CentOS 7 需 `yum install -y net-tools` 安装。

| 命令 | 作用 |
| --- | --- |
| `ifconfig` | 查看已启用的接口 |
| `ifconfig -a` | 查看所有接口（含未启用） |
| `ifconfig eth0 up` / `down` | 启用 / 停用 |
| `ifconfig eth0 10.0.0.10 netmask 255.255.255.0` | 临时配置 IP |
| `ifconfig eth0:0 10.0.0.11/24` | 配置 IP 别名（虚拟 IP） |
| `ifconfig eth0 mtu 9000` | 设置 MTU |
| `ifconfig eth0 promisc` / `-promisc` | 开启/关闭混杂模式 |

## ethtool 速查

ethtool 用于查看和配置网卡**物理层**参数，排查带宽、丢包、网卡 offload 等问题时不可或缺。CentOS 7 默认安装，若缺失可 `yum install -y ethtool`。

| 命令 | 作用 |
| --- | --- |
| `ethtool eth0` | 查看网卡基础信息（速率、双工、Link 状态） |
| `ethtool -i eth0` | 查看驱动、固件版本 |
| `ethtool -S eth0` | 查看网卡收发包详细统计（丢包、CRC 错误等） |
| `ethtool -g eth0` | 查看 ring buffer 大小 |
| `ethtool -G eth0 rx 4096 tx 4096` | 设置 ring buffer（调优高并发） |
| `ethtool -k eth0` | 查看 offload 状态（tso/gso/gro/lro） |
| `ethtool -K eth0 gro off tso off` | 关闭指定 offload（调试丢包常用） |
| `ethtool -l eth0` | 查看支持的队列数（多队列） |
| `ethtool -L eth0 combined 4` | 设置组合队列数 |
| `ethtool -m eth0` | 读取 SFP 光模块信息（DOM） |

### 查看网卡详情

```bash
ethtool eth0
```

输出示例：

```text
Settings for eth0:
    Supported ports: [ TP ]
    Supported link modes:   1000baseT/Full
                            10000baseT/Full
    Supported pause frame use: No
    Supports auto-negotiation: Yes
    Advertised auto-negotiation: Yes
    Speed: 10000Mb/s
    Duplex: Full
    Port: Twisted Pair
    PHYAD: 0
    Transceiver: internal
    Auto-negotiation: on
    Link detected: yes
```

> 持久化配置：ethtool 修改的参数重启后失效，建议写入 `/etc/sysconfig/network-scripts/ifcfg-eth0` 的 `ETHTOOL_OPTS` 参数或使用 systemd unit。
