# 配置中文输入法

Kali Linux 安装中文输入法的完整步骤,涵盖 fcitx5(推荐)与 ibus(备选)两套方案,以及环境变量配置与常见问题排查。

## 桌面环境与输入法框架

Kali 默认桌面环境在 X11 会话下为 XFCE 或 GNOME(取决于安装镜像),两者都支持主流输入法框架:

| 框架 | 特点 | 推荐场景 |
| --- | --- | --- |
| **fcitx5** | 现代、轻量、活跃维护,生态丰富 | 首选方案 |
| ibus | GNOME 默认集成,稳定轻量 | GNOME 桌面 / 极简环境 |

## 方案一:fcitx5(推荐)

### 1. 安装 fcitx5 与中文拼音引擎

```bash
sudo apt update
sudo apt install -y fcitx5 fcitx5-chinese-addons fcitx5-frontend-gtk3 fcitx5-config-qt
```

### 2. 配置环境变量

编辑 `~/.xprofile`(单用户)或 `/etc/environment`(全局),写入以下内容:

```bash
GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx
SDL_IM_MODULE=fcitx
GLFW_IM_MODULE=ibus
```

### 3. 切换默认输入法框架并重启

```bash
im-config -n fcitx5
reboot
```

登录后在系统托盘右键 fcitx5 图标 → **Configure** → 添加 **Pinyin** 输入法。

### 4. 切换快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Ctrl+Space` | 中英文切换(默认) |
| `Ctrl+Shift` | 切换输入法 |
| `Super+Space` | 切换输入法(GTK 默认) |

## 方案二:ibus(备选)

```bash
sudo apt install -y ibus ibus-pinyin
im-config -n ibus
reboot
```

登录后执行 `ibus-setup` 添加"拼音"输入法,`Super+Space` 切换。

## 常见问题

| 问题 | 解决办法 |
| --- | --- |
| 安装后无图标 | 检查 `echo $GTK_IM_MODULE` 是否为 `fcitx`,确认已注销重新登录 |
| 中文显示为方块 | 安装中文字体:`sudo apt install -y fonts-noto-cjk fonts-wqy-zenhei` |
| Chrome/VS Code 无法输入中文 | 确认 `GTK_IM_MODULE` 与 `QT_IM_MODULE` 已设置;VS Code 可加 `--enable-features=UseOzonePlatform --ozone-platform=wayland` |
| Wayland 会话下不工作 | 改用 X11 会话登录;fcitx5 在 Wayland 下需额外配置 |
| locale 未生成中文 | 执行 `sudo dpkg-reconfigure locales` 勾选 `zh_CN.UTF-8`,再重启会话 |
