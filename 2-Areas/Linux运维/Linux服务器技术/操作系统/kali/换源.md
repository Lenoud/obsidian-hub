# Kali 换源

Kali 采用 **rolling**(滚动发布)模型,所有软件包都从 `kali-rolling` 分支获取。官方源位于国外,国内访问较慢,建议替换为国内镜像以显著提速。

## sources.list 格式

软件源配置文件路径:

```text
/etc/apt/sources.list
```

Kali 的 `sources.list` 行格式:

```text
deb <URL> kali-rolling main contrib non-free non-free-firmware
```

| 字段 | 含义 |
| --- | --- |
| `deb` | 二进制包(必需) |
| `deb-src` | 源码包(可选,仅编译/调试需要) |
| `kali-rolling` | Kali 唯一的滚动分支 |
| `main` | DFSG 自由软件 |
| `contrib` | 依赖非自由软件的自由软件 |
| `non-free` | 非自由软件 |
| `non-free-firmware` | 非自由固件(2023 年起单独拆分) |

> **注:** 自 Kali **2020.4** 起默认不再带 `deb-src`,除非需要重新编译或调试,否则无需添加。

## 国内镜像源(任选其一)

直接覆盖写入 `/etc/apt/sources.list`:

```bash
# 阿里云(推荐,带宽大)
deb https://mirrors.aliyun.com/kali kali-rolling main non-free contrib

# 中科大 USTC
deb http://mirrors.ustc.edu.cn/kali kali-rolling main non-free contrib

# 清华大学 TUNA
deb http://mirrors.tuna.tsinghua.edu.cn/kali kali-rolling main contrib non-free

# 浙江大学 ZJU
deb http://mirrors.zju.edu.cn/kali kali-rolling main contrib non-free

# 东软 Neusoft
deb http://mirrors.neusoft.edu.cn/kali kali-rolling main non-free contrib

# 重庆大学 CQU
deb http://mirrors.cqu.edu.cn/kali kali-rolling main non-free contrib
```

## 更新与升级

```bash
# 刷新软件索引并执行完整滚动升级
sudo apt update
sudo apt full-upgrade -y

# 清理残留依赖
sudo apt autoremove --purge -y
```

## 验证当前源

```bash
# 查看某个包从哪个源获取
apt-cache policy <package>

# 查看系统版本
lsb_release -a
```

## 注意事项

- **不要混用发行版源**:Kali 基于 Debian Testing,但不要直接套用 Debian 的源,否则会破坏依赖关系。
- **HTTPS 源需安装证书**:`sudo apt install -y apt-transport-https ca-certificates`。
- **Kali 官方源(`http.kali.org`)在国外**,国内访问缓慢,建议优先使用上述国内镜像。
- **企业内网**:可搭建本地 apt-cacher 或 nexus 代理缓存 Kali 源。
