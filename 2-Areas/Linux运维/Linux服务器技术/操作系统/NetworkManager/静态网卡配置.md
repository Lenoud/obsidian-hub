# 静态网卡配置

使用 NetworkManager 的 nmtui 或 nmcli 配置静态网卡。

## GUI配置网卡
可以使用箭头键或按 Tab 键向前选择选项，按 Shift+Tab 组合键返回。按 Enter 选择一个选项。按 Space 键选择复选框状态。

#nmtui提供基于光标的文本用户界面（TUI）

#nmtui可提供一个文本界面来与NetworkManager交互用于配置网络。

#该工具包含在NetworkManager-tui子软件包中。

#通常不会默认随NetworkManager的安装而安装该子软件包。

#需要单独安装 NetworkManager-tui。

```shell
nmtui  #使用gui界面配置网络设置
```

## 静态配置文件模板
```shell
ens33.nmconnection
[connection]
id=ens33
uuid=83fe92c8-bc98-3267-b683-78a521e8a1f6
type=ethernet
autoconnect-priority=-999
interface-name=ens33
timestamp=1720746499

[ethernet]

[ipv4]
address1=192.168.100.120/24,192.168.100.2
dns=114.114.114.114;
method=manual

[ipv6]
addr-gen-mode=eui64
method=auto

[proxy]
```

## 重启网卡
通过操作NetworkManager也可以修改网络配置

```shell
 systemctl restart NetworkManager
```

使用nmcli重启网卡，service和systemctl在centos 9 stram中已经不能操作网卡

```shell
nmcli c reload                         # 重新加载配置文件
nmcli c up ens160                      # 重启ens33网卡
```
