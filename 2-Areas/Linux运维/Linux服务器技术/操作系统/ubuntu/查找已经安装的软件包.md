# 查找已经安装的软件包

在 Ubuntu / Debian 系统中查询已安装软件包的多种方法。

## dpkg 方式

```bash
# 列出所有已安装的包(最全面)
dpkg --get-selections

# 过滤某个包(查 nginx 是否装了)
dpkg --get-selections | grep nginx
# nginx-common                          install
# nginx-core                            install

# 查看某个包的详细信息(版本、依赖、文件列表)
dpkg -l nginx
dpkg -s nginx                # 详细信息
dpkg -L nginx                # 这个包安装了哪些文件

# 反查:某个文件是哪个包装的
dpkg -S /usr/bin/curl
# curl: /usr/bin/curl
```

## apt 方式

```bash
# 列出所有包(新版 apt,输出比 dpkg 更易读)
apt list --installed

# 过滤
apt list --installed | grep -i python

# 只列出有更新可用的包
apt list --upgradable

# 列出某个包的所有版本(含可安装版本)
apt-cache policy nginx
```

## snap 方式

```bash
# 列出所有 snap 应用
snap list
```

## 根据文件反查(跨包管理器)

```bash
# 已知命令路径,反查属于哪个包
which curl                              # /usr/bin/curl
dpkg -S /usr/bin/curl                   # curl: /usr/bin/curl

# 如果是 snap / pip / npm 装的,dpkg 找不到,用对应工具
snap find curl
pip list | grep -i curl
```

## 统计与导出

```bash
# 统计已安装包总数
dpkg --get-selections | wc -l

# 导出当前已安装列表(用于迁移/重建)
dpkg --get-selections > installed-packages.txt

# 在新机器上导入并安装
dpkg --clear-selections
sudo dpkg --set-selections < installed-packages.txt
sudo apt-get dselect-upgrade
```
