# 配置中文输入法

Ubuntu 系统配置中文输入法（fcitx / sogou）。

```bash
#如果安装了sogou不能用，卸载 fcitx sogou 输入法

# 彻底卸载sogou和fcitx
sudo apt purge sogoupinyin
sudo apt purge fcitx
sudo apt autoremove
sudo apt remove fcitx*
sudo apt purge fcitx*
sudo apt autoremove

# 删除配置文件
cd .config/
rm -rf sogoupinyin/
rm -rf ibus

安装&配置 ibus-rime
# 安装ibus
sudo apt install ibus ibus-rime

这里需要先注销或重启下系统，然后进入输入法配置

[设置] - [区域与语言] - [输入源] - [+]

选择 中文(Rime)

然后删掉其他不需要的输入法

[管理已安装的语言] - [键盘输入法系统]

选择ibus

#重启ibus
ibus restart
#原文链接：https://blog.csdn.net/qq_44866828/article/details/122472530
```
