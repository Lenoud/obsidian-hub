# awk

使用awk命令输出文本文件的第三列，可以使用以下命令：

```bash
awk '{print $3}' 文件名
```

常用操作包括：

1. 分隔符：使用-F选项指定分隔符，默认使用空格或Tab作为分隔符。
2. 打印行：使用print命令打印输出，如awk '{print $1}' file.txt表示打印第一列的内容。
3. 条件语句：使用if语句进行条件判断，如awk '{if($1>10) print $0}' file.txt表示如果第一列的值大于10就打印输出整行。
4. 循环语句：使用for语句进行循环操作，如awk '{for(i=1;i<=NF;i++) print $i}' file.txt表示对每一行进行遍历并打印每一列的内容。
5. 内部变量：AWK内置了许多变量，如$0表示整行内容，$1表示第一列内容，NF表示当前行的字段数量等等。
