# sed

sed 流编辑器的使用方法。

```bash
sed -i.bak 's/原字符串/新字符串/g' file.txt
#以上命令将在执行替换操作前备份 file.txt 文件为 file.txt.bak
```

1. s 命令：替换命令，可以用于替换指定文本。
    - 语法：s/原字符串/新字符串/g，其中s表示substitute（替换），g表示全局替换。
    - 示例：将文本中的 foo 替换成 bar，可以使用 sed 's/foo/bar/g' file.txt 命令。
2. d 命令：删除命令，可以用于删除指定行或匹配到的行。
    - 语法：d，表示删除当前行；/[匹配字符串]/d，表示删除匹配到的行。
    - 示例：删除文件中所有包含 foo 的行，可以使用 sed '/foo/d' file.txt' 命令。
3. i 命令：添加命令，可以用于在指定行前添加新的内容。
    - 语法：i 新文本，表示在当前行前添加一行新文本。
    - 示例：在文件第一行添加新文本 Hello World，可以使用 sed '1 i Hello World' file.txt' 命令。
4. a 命令：添加命令，可以用于在指定行后添加新的内容。
    - 语法：a 新文本，表示在当前行后添加一行新文本。
    - 示例：在文件最后一行添加新文本 Goodbye，可以使用 sed '$ a Goodbye' file.txt' 命令。
5. p 命令：打印命令，可以用于打印指定行或匹配到的行。
    - 语法：p，表示打印当前行；/[匹配字符串]/p，表示打印匹配到的行。
    - 示例：打印文件所有包含 foo 的行，可以使用 sed -n '/foo/p' file.txt' 命令。
