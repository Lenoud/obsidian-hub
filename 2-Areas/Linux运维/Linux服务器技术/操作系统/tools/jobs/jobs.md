# jobs

要将一个后台作业调到前台，可以使用以下两个命令：

jobs：该命令用于列出当前在后台运行的作业，并显示每个作业的标识符（job ID）。

fg：该命令用于将一个后台作业调到前台运行。可以使用作业的标识符作为参数，例如fg %1表示将标识符为1的作业调到前台。

下面是具体的步骤：

使用jobs命令查看后台作业的列表和标识符。

使用fg命令将所需作业调到前台。例如，如果要将标识符为1的作业调到前台，则可以运行fg %1。

这样，后台作业就会切换到前台，并在终端中显示其输出。

要将当前正在前台运行的作业调到后台，可以使用以下两个命令：

暂停当前正在前台运行的作业：按下 Ctrl + Z 键来暂停当前作业，并将其放在后台等待。

将作业切换到后台继续运行：使用 bg 命令将暂停的作业切换到后台继续运行。例如，运行 bg %1 将标识符为 1 的作业切换到后台。

这样，作业就会继续在后台运行，你可以继续在终端中执行其他命令。注意，在后台运行的作业将不会在终端中显示输出。若需要查看输出，可以使用输出重定向或者使用 fg 命令将作业切换回前台
