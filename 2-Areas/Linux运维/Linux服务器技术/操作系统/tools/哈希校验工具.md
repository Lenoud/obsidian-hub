# 哈希校验工具

文件哈希校验工具（md5sum、sha256sum 等）的使用方法。

## linux平台下
### 1.openssl
openssl dgst -[算法] [文件路径]

```text
openssl dgst -sha256 filename.txt
openssl dgst -md5 filename.txt
```

## windows平台下
### 1.certutil
在 Windows 上，certutil 是一个强大的工具，用于处理各种与证书相关的任务。

```text
certutil -hashfile  .\.heyboxacc.conf md5
certutil -hashfile  .\.heyboxacc.conf sha256
```
