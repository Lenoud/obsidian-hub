# 域名相关

域名解析相关命令（nslookup、dig 等）。

| **命令** | 功能描述 | 常用参数 | 使用方式 |
| :---: | :---: | :---: | :---: |
| **host** | 根据域名查询IP地址和反向解析 | 无 | host example.com |
| **nslookup** | 查询域名对应的IP地址和其他DNS记录 | 无 | nslookup example.com |
| **dig** | DNS信息查询工具，支持多种查询类型，并提供详细的返回结果 | +trace：沿着查找路径显示DNS解析过程
+short：以简短格式显示结果
-x IP：进行反向解析 | dig example.com
dig +trace example.com |
| **whois** | 查询域名、IP地址或AS号码的注册信息 | 无 | whois example.com |

以上是关于 host、nslookup、dig 和 whois 四个命令的介绍、常用参数和使用方式的表格形式总结：

+ host 命令用于查询域名对应的IP地址和反向解析信息；
+ nslookup 命令用于查询域名对应的IP地址和其他DNS记录；
+ dig 命令是一个功能强大的DNS信息查询工具，可以支持多种查询类型，并提供详细的返回结果；
+ whois 命令用于查询域名、IP地址或AS号码的注册信息。
