# nmap网络扫描工具

nmap 网络扫描与安全审计工具。

```bash
#查询所有端口
nmap -p- -v [目标IP地址]
```

面是有关nmap命令的详细介绍，以及一些常用参数和示例：

| 参数 | 描述 | 示例 |
| --- | --- | --- |
| -sS | TCP SYN 扫描 | nmap -sS 192.168.1.1 |
| -sT | TCP 连接扫描 | nmap -sT 192.168.1.1 |
| -sU | UDP 扫描 | nmap -sU 192.168.1.1 |
| -sN | TCP Null 扫描 | nmap -sN 192.168.1.1 |
| -sF | TCP FIN 扫描 | nmap -sF 192.168.1.1 |
| -sX | TCP Xmas Tree 扫描 | nmap -sX 192.168.1.1 |
| -sP | Ping 扫描 | nmap -sP 192.168.1.0/24 |
| -sV | 服务版本检测 | nmap -sV 192.168.1.1 |
| -O | 操作系统检测 | nmap -O 192.168.1.1 |
| -A | 所有信息扫描 | nmap -A 192.168.1.1 |
| -p | 指定端口号进行扫描 | nmap -p 80,443 192.168.1.1 |
| -F | 快速扫描，只扫描常见端口 | nmap -F 192.168.1.1 |
| -v | 显示详细信息 | nmap -v 192.168.1.1 |
| -O | 输出所有已经扫描过的主机 | nmap -oN output.txt 192.168.1.0/24 |
