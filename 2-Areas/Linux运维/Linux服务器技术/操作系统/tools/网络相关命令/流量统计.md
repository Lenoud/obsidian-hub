# 流量统计

ifstats 和 iftop 两个命令的详细介绍、常用参数以及使用方式的表格形式总结：

| **命令** | **功能描述** | **常用参数** | **使用方式** |
| :---: | :---: | --- | --- |
| **ifstat** | 显示网络接口的实时流量统计信息 | -a：显示所有接口的统计信息
-l：以连续模式显示流量统计信息
-z：将流量统计信息归零
-n：禁用反向DNS解析
-v：显示版本信息
-h：显示帮助信息
-t：显示时间戳
-i  if0,if1,...：指定要监测的网络接口 | ifstat
ifstat -a
ifstat -l -i eth0
ifstat -z -i wlan0 |
| **iftop** | 用于实时监测网络流量，显示当前网络连接的带宽使用情况，以及每个连接的发送和接收速率等信息 | -i：指定要监测的网络接口
-n：禁用反向DNS解析
-F：显示完整的域名（默认只显示主机名部分）
-P 显示端口号 | iftop
 iftop -i eth0
 iftop -i eth0 -P -B
iftop -B
iftop -i eth0 -P -B -f 'port 25565' |
| **nload** | nload是一个用于显示网络流量的命令行工具，它可以实时监测网络接口的流量使用情况。 | -t <seconds>：设置刷新间隔时间，单位为毫秒。
-U <unit>：设置流量显示单位，可选项包括Bit、Byte、KB、MB、GB等。
-u <speed>：设置带宽上限值，超过该值的流量将以红色显示。
-m：不显示使用情况。
-h：显示帮助信息，列出所有可用参数和选项。 | nload  eth0 -t 1000 -U MB -u M

 |
