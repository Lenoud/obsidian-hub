# 网络连接情况

查看服务器网络连接状态的方法。

| **命令** | **功能描述** | **常用参数** | **使用方式** |
| :---: | :---: | :---: | --- |
| **netstat** | 用于显示与 IP、TCP、UDP 和 ICMP 协议相关的统计数据，以及打开的网络连接和监听端口等信息 | -a：显示所有连接和监听端口
-n：禁用反向DNS解析
-p：显示与给定进程ID或名称相关的连接 | netstat -ano
netstat -antp |
| **ss** | 超级 netstat，可以列出更多协议、状态等信息 | -a：显示所有连接和监听端口
-n：禁用反向DNS解析
-p：显示与给定进程ID或名称相关的连接
-t：仅显示TCP连接
-u：仅显示UDP连接 | ss -ano
ss -antp
 |
| **lsof** | 列出系统打开文件的相关信息，可以查看当前文件被哪些进程占用，以及进程正在使用哪些文件等信息 | -i：仅显示网络连接相关信息
-n：禁用反向DNS解析
-p：显示与给定进程ID或名称相关的文件 | lsof -i
lsof -i TCP:80 |
