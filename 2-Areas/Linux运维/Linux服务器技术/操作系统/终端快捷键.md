# 终端快捷键

Linux 终端常用快捷键速查。



# 最常用的快捷键
| Ctrl + F | 光标向前移动一个字符 |
| --- | --- |
| Ctrl + B | 光标向后移动一个字符 |
| Ctrl + N | 查看下一个命令 |
| Ctrl + P | 查看上一个命令 |
| Ctrl + L | 清屏 |

| Ctrl + A | 光标移到行首 |
| --- | --- |
| Ctrl + E | 光标移到行尾 |
| Ctrl + U | 删除光标之前的所有内容 |
| Ctrl + K | 删除光标之后的所有内容 |
| Ctrl + L | 清屏 |
| Ctrl + R | 向后搜索命令历史 |
| Ctrl + C | 中断当前进程 |
| Ctrl + Z | 暂停当前进程 |
| Ctrl + D | 退出当前终端或者结束输入（如果在空行） |
| Ctrl + W | 删除光标之前的一个单词 |
| Ctrl + Y | 粘贴之前删除的文本 |
| Ctrl + T | 交换光标处字符和前一个字符位置 |
| Alt + B | 光标倒退一个单词 |
| Alt + F | 光标前进一个单词 |
| Ctrl + _ | 撤销上一次操作 |
| Ctrl + X, Ctrl + E | 打开文本编辑器编辑当前命令 |

| 快捷键 | 描述 |
| --- | --- |
| Ctrl + D | 输入文件结束符，相当于输入 EOF |
| Ctrl + Z | 将当前进程移到后台，并暂停它 |
| Ctrl + R | 在历史命令中向后搜索 |
| Ctrl + S | 停止输出到终端 |
| Ctrl + Q | 恢复输出到终端 |
| Ctrl + T | 交换光标前后两个字符位置 |
| Ctrl + F | 光标向前移动一个字符 |
| Ctrl + B | 光标向后移动一个字符 |
| Ctrl + H | 删除光标前一个字符 |
| Ctrl + N | 查看下一个命令 |
| Ctrl + P | 查看上一个命令 |
| Ctrl + L | 清屏 |
| Ctrl + W | 删除光标前一个单词 |
| Ctrl + K | 删除光标后的所有内容 |
| Ctrl + U | 删除光标前的所有内容 |
| Ctrl + Y | 粘贴最后一次删除的内容 |
| Ctrl + C | 终止当前命令 |
| Ctrl + X, Ctrl + E | 打开默认的文本编辑器来编辑当前命令 |
