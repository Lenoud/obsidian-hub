# nano快捷键

Nano 文本编辑器常用快捷键速查。

**注意**：nano中，黑底白字表示快捷键操作。其中“^”表示Ctrl键，则Ctrl + G就表示成“^G”。“M”表示Alt键，则Alt + W表示为“M-W”。

| 快捷键 | 描述 |
| --- | --- |
| Ctrl + G | 帮助（显示快捷键列表） |
| Ctrl + X | 退出编辑器（如果有未保存修改则会提示保存） |
| Ctrl + O | 保存文件（如果是新文件，会提示输入文件名） |
| Ctrl + W | 查找（输入要查找的文本，再按一次查找下一个） |
| Ctrl + K | 剪切当前行或选中的文本 |
| Ctrl + U | 粘贴剪切或复制的文本 |
| Ctrl + C | 显示当前光标位置 |
| Ctrl + \ | 替换文本 |
| Ctrl + Shift + 6 | 开始选择文本 |
| Alt + A | 开启/关闭自动换行 |
| Alt + < | 移动到文件开头 |
| Alt + > | 移动到文件末尾 |
| Alt + / | 函数块的括号匹配 |

**搜索和替换**：按下 Ctrl + W 进行搜索，按下 Ctrl + \ 进行替换。

**移动光标**：使用方向键可以移动光标。此外，你也可以使用 Ctrl + Space 来设置标记，然后使用方向键来扩展选择区域。

**多个缓冲区**：在 Nano 中，你可以同时打开多个文件，并且可以使用 Ctrl + ^ 来切换缓冲区。

**语法高亮**：Nano 默认不会进行语法高亮显示，但是你可以通过在命令行中使用 nano -Y <语言> 来启用相应语言的语法高亮功能。

**插入文件内容**：在编辑过程中，可以使用 Ctrl + R 来插入另一个文件的内容，这对于需要粘贴其他文件的内容非常有用。
