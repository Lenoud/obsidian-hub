# 创建虚拟机

KVM 虚拟机的创建方法。

```bash
virt-install \
--connect qemu:///system \
--name windows-server-2012-r2 \
--ram 2048 \
--vcpus 4 \
--network network=default,model=virtio \
--disk path=ISO系统文件,device=cdrom,bus=ide \
--disk path=virtio驱动文件,device=cdrom,bus=ide \
--disk path=磁盘文件,format=qcow2,device=disk,bus=virtio \
--os-type windows \
--os-variant win2k12r2 \
--vnc
```
